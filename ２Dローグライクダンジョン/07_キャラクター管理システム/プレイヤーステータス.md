## 1. ステータス一覧

### 1-1. 基本ステータス（Base Stats）

| ステータス名 | 略称 | 説明 |
|---|---|---|
| **最大HP** | MaxHP | ダメージを受けると減少。0で行動不能（帰還）。 |
| **現在HP** | HP | 現在の体力。施設や食料で回復。 |
| **最大MP** | MaxMP | スキル・魔法の使用コスト。ダンジョン内では宿泊・特定アイテムで回復。 |
| **現在MP** | MP | 現在の魔力残量。 |
| **霊素（空腹度）** | Spirit | ダンジョン内専用のスタミナ。毎ターン消費する。0になると毎ターンHP減少。 |
| **攻撃力** | ATK | 物理攻撃時のダメージ基礎値。 |
| **魔法攻撃力** | MAG | 魔法スキル・魔法攻撃のダメージ基礎値。 |
| **防御力** | DEF | 物理ダメージの軽減値。 |
| **魔法防御力** | MDF | 魔法ダメージの軽減値。 |
| **精神** | MND | 状態異常への耐性・霊素0時のHP減少軽減・MP微量回復に関与する。詳細は後述。 |
| **素早さ** | SPD | 行動順に影響（将来の複数キャラ対応時）。回避率にも一部寄与。 |
| **幸運** | LUK | アイテムのドロップ率・鑑定成功率・ミニゲームの判定に影響。 |

### 1-2. 派生ステータス（計算で求める値）

| ステータス名 | 計算式（案） | 説明 |
|---|---|---|
| **物理ダメージ** | `ATK + 武器ATK - 敵DEF` （最低1） | 通常攻撃の最終ダメージ |
| **魔法ダメージ** | `MAG + スキル倍率 - 敵MDF` （最低1） | 魔法スキルの最終ダメージ |
| **回避率** | `SPD × 0.2 + LUK × 0.1`（%、上限30%) | 攻撃をかわせる確率 |
| **クリティカル率** | `LUK × 0.3`（%、上限40%） | クリティカルヒット発生確率 |
| **クリティカル倍率** | 固定 `× 1.5`（装備・スキルで上昇可） | クリティカル時のダメージ倍率 |
| **状態異常耐性** | `MND × 0.5`（%、上限50%） | 各状態異常の付与成功率を減少させる |
| **霊素底力** | `MND × 0.05`（HP減少軽減量、上限-2） | 霊素0時のHP減少量を軽減する（詳細は霊素システム参照） |
| **MP自然回復** | `MND ÷ 20`（端数切り捨て）/10ターン | ダンジョン内で精神が高いほどMPが微量回復する |

## 2. 霊素（Spirit）システム

> **設定的意味:** プレイヤーの霊魂が現世の鎧に留まるためのエネルギー。消耗すると霊魂が不安定になる。

### 2-1. 基本仕様

| 項目 | 値（案） | 備考 |
|---|---|---|
| **初期最大霊素** | 100 | レベルアップや施設強化で増加しない（食料アイテムで管理） |
| **最大霊素の上限** | 200 | 特定の施設強化・ジョブスキルで解放 |
| **消費タイミング** | 1ターン行動するごと | 移動・攻撃・アイテム使用すべて1ターン |
| **1ターンあたりの消費量** | -1 （基本） | ⚠️ 未確定。快適性とのトレードオフ要調整 |
| **霊素0の効果** | 毎ターン HP -3（基本値） | MND による軽減あり。`実際の減少 = 3 - (MND × 0.05)` 最低1 ⚠️ 数値要調整 |
| **霊素0による帰還** | なし（HPが0になるまで続ける） | ゲームオーバーにはならない仕様 |

### 2-2. 霊素の回復手段

| 手段 | 回復量 | 入手場所 |
|---|---|---|
| 霊素食料（小） | +20 | ダンジョン内ドロップ・商店 |
| 霊素食料（中） | +40 | ダンジョン深層・クラフト |
| 霊素食料（大） | +60 | 拠点の料理屋・農場産 |
| 聖泉（フロア内） | 全回復 | 特定フロアのギミック（1回限り） |
| 宿屋（拠点） | 全回復 | ダンジョン外でのみ利用可能 |

## 3. レベルアップ仕様

### 3-1. 経験値テーブル（案）

```
次レベルまでの必要経験値 = 50 × LV^1.5
（例: LV1→2: 50, LV5→6: 559, LV10→11: 1581）
```

> ⚠️ 係数は未確定。ゲームテンポに応じて調整が必要。

### 3-2. レベルアップ時の処理

1. 下記のステータスが**自動的に増加**する。
2. プレイヤーへレベルアップ演出（テキストログ + 簡易エフェクト）を表示。
3. **HPとMPは全回復しない**（ダンジョン内での安全なレベリングを防ぐ）。

### 3-3. レベルアップ時のステータス増加量（案）

| ステータス | 増加量/レベル | 備考 |
|---|---|---|
| MaxHP | `+8 + クラス補正` | 戦士系は多め、魔法系は少なめ |
| MaxMP | `+4 + クラス補正` | 魔法系は多め |
| ATK | `+2 + クラス補正` | |
| MAG | `+2 + クラス補正` | |
| DEF | `+1 + クラス補正` | |
| MDF | `+1 + クラス補正` | |
| MND | `+1 + クラス補正` | 僧侶・魔法使い系は多め |
| SPD | `+0.5`（端数切り捨て） | |
| LUK | `+0.5`（端数切り捨て） | |

### 3-4. 職業（ジョブ）クラス補正（乗算係数 案）

| ステータス | 戦士 | 魔法使い | 僧侶 | 盗賊 | 弓使い |
|---|---|---|---|---|---|
| MaxHP | × 1.5 | × 0.8 | × 1.1 | × 1.0 | × 1.0 |
| MaxMP | × 0.7 | × 1.5 | × 1.3 | × 0.9 | × 0.8 |
| ATK | × 1.3 | × 0.8 | × 0.9 | × 1.1 | × 1.2 |
| MAG | × 0.7 | × 1.5 | × 1.2 | × 0.8 | × 0.8 |
| DEF | × 1.3 | × 0.7 | × 1.0 | × 1.0 | × 1.0 |
| MND | × 0.8 | × 1.2 | × 1.5 | × 0.9 | × 1.0 |
| SPD | × 0.9 | × 1.0 | × 0.9 | × 1.3 | × 1.1 |
| LUK | × 1.0 | × 1.0 | × 1.1 | × 1.3 | × 1.2 |

> **注:** 転職時、上位職の補正は基本職の補正に**累積加算**ではなく**上書き**する方式を想定。詳細は `../06_職業_ビルド構築システム/職業システム概要案.md` を参照。

## 4. 状態異常（ステータス異常）

### 4-1. 異常状態一覧

| 状態異常名 | 効果 | 解除条件 |
|---|---|---|
| **毒** | 毎ターン HP -3〜5 | 解毒草・階段を降りる |
| **麻痺** | 2ターンに1回しか行動できない | 3ターン経過・解麻痺草 |
| **混乱** | 行動方向がランダムになる | 3ターン経過・特定アイテム |
| **眠り** | 行動不能。ダメージを受けると解除 | ダメージを受ける・特定ターン経過 |
| **呪い** | 装備アイテムが外せなくなる | 呪い解除の巻物・特定施設 |
| **霊素加速** | 霊素消費が2倍になる（バッドステータス） | 特定ターン経過・回復アイテム |
| **属性弱体** | 特定属性ダメージを 1.5倍受ける | 階段を降りる・特定アイテム |

### 4-2. 精神（MND）による耐性

状態異常の付与判定は以下の式で行う。

```
付与成功率 = 基本付与率 × (1 - MND × 0.005)
（例: MND=20 なら 10% 軽減、MND=50 なら 25% 軽減、上限50%）
```

- **呪い**は精神耐性の影響を受けない（装備に直接かかる呪いのため）。
- **混乱・眠りなどの精神系異常**は MND の影響を特に強く受ける（軽減係数 × 1.5 倍）。

### 4-3. 重複ルール

- 同じ状態異常は重複しない（再付与でターン数がリセット）。
- 毒と麻痺など**異なる異常は同時に複数付与可能**。

## 5. 死亡・敗北処理

| ケース | 処理 |
|---|---|
| HP が 0 になった | 「帰還演出」→ 町（セーフゾーン）に戻る |
| 持ち越しアイテム | **ダンジョン持込アイテムはすべてロスト**（ローグライトルール） |
| 経験値・レベル | **保持**（レベルはリセットされない） |
| 救出・捕獲済みの霊魂/魔物 | **保持**（ダンジョン内で出会い、旗を立てた対象のみ） |
| 拠点の装備・素材 | **保持**（町に預けたものは失わない） |
