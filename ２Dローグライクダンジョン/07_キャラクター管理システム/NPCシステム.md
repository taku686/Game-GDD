# NPCシステム 仕様

> **関連ファイル:** `概要書.md` / `../04_町育成_拠点システム/各種施設一覧とその効果.md` / `エネミーAI.md`

## 1. 概要と登場人物の分類

本作のNPCは大きく **「霊魂（Spirit）」** と **「魔物（Monster）」** の2種類に分かれる。
どちらも**ダンジョン内で出会い、拠点（町）に連れ帰る**ことでゲームの循環に組み込まれる。

| 種別 | 設定 | 主な役割 |
|---|---|---|
| **霊魂（Spirit）** | かつて生きていた者の魂。依り代（鎧・人形）に憑依している | 救出 → 拠点の住人 → 施設に配置してボーナス付与 |
| **魔物（Monster）** | ダンジョンを徘徊する魔の存在 | 捕獲 → 拠点の労働力 or ダンジョン同行ペット |

## 2. 霊魂（Spirit）の救出フロー

### 2-1. 発見と遭遇

1. 特定フロアに「囚われた霊魂」が配置される（マップ自動生成時に配置）。
2. プレイヤーが隣接マスに移動し、「話しかける」コマンドを選択する。
3. 短い会話イベントが発生し、以下の選択肢が現れる。

```
[会話イベント]
  ├─ 「救出する」  → 解放クロスタル消費 → 仮登録（同行 or 後で拠点へ）
  ├─ 「後でくる」  → スポット記憶。次回同じフロアに来た際に残っている
  └─ 「立ち去る」  → 何もしない
```

### 2-2. 解放クロスタル（救出コスト）

- 霊魂を解放するには「解放クロスタル」（専用アイテム）が必要。
- ダンジョン内で入手できるほか、拠点の商店でも購入可能。
- 霊魂のレアリティや職種に応じてコストが変わる。

| レアリティ | 必要クロスタル数（案） |
|---|---|
| ★1（コモン） | 1個 |
| ★2（アンコモン） | 2個 |
| ★3（レア） | 3個 |
| ★4（エピック） | 5個 |

### 2-3. 救出後の処理

| 選択肢 | 処理 |
|---|---|
| **同行させる（ペット）** | 後述のペットシステムで一緒に探索する |
| **先に拠点へ送る** | その場で拠点登録。プレイヤーの探索を継続できる（推奨） |

---

## 3. 魔物（Monster）の捕獲フロー

### 3-1. 捕獲条件

魔物を捕獲するには、**HPをある程度削った上で捕獲アイテムを使用する**必要がある。

```
①  対象魔物のHPを [最大HP × 30%以下] まで削る
②  「捕獲の鎖」（捕獲専用アイテム）を使用
③  成功率判定（下記参照）
    成功 → 捕獲完了（同行 or 拠点送り）
    失敗 → 魔物が逃走ステートに移行
```

### 3-2. 捕獲成功率

```
捕獲成功率 = (最大HP - 現HP) / 最大HP × 70% + LUK × 0.5%
（上限: 90%）
```

> ⚠️ 数値は未確定。調整が必要。

### 3-3. 捕獲後の処理

霊魂と同様、捕獲後は「同行」か「拠点送り」を選択できる。

## 4. 拠点登録後の管理

### 4-1. 霊魂の施設配置

拠点に登録した霊魂は**特定の施設に「配置」**することでボーナスが発生する。

| 配置先施設 | 配置可能な霊魂タイプ | ボーナス例 |
|---|---|---|
| 鍛冶屋 | 元・戦士/鍛冶師タイプ | 強化成功率 +5%、強化コスト -10% |
| 商店 | 元・商人タイプ | 売値 +10%、レアアイテム出現率 +5% |
| 宿屋/ギルド | 元・冒険者/聖職者タイプ | HP回復量 +20%、クエスト報酬 +10% |
| 料理屋 | 元・料理人タイプ | 霊素回復アイテムの効果 +20% |
| 魔導研究所 | 元・魔法使い/学者タイプ | 魔法スキルのMP消費 -15% |

> 詳細な施設ごとのボーナス値は `../04_町育成_拠点システム/各種施設一覧とその効果.md` に委ねる。

### 4-2. 霊魂のレアリティと個体差

各霊魂はレアリティに加え、**個体固有のパラメータ**を持つ。

| パラメータ | 説明 |
|---|---|
| **職種（ジョブ）** | 鍛冶師・商人・農民・魔法使いなど。配置できる施設が決まる |
| **親密度** | プレイヤーとの交流で上昇。一定値でボーナスが強化される |
| **特技** | 霊魂固有のパッシブボーナス（例：「倹約家 → 建築コスト -5%」） |
| **依り代** | 憑依している鎧・人形の見た目。着せ替えによって変化可 |

### 4-3. 魔物の牧場管理

拠点に登録した魔物は**「魔物牧場」施設**で管理される。

| 役割 | 内容 |
|---|---|
| **労働力** | 牧場に配置して資源（素材・食料）を自動生産する |
| **ペット待機** | ダンジョン同行のために待機させる |
| **繁殖（将来検討）** | 同種の魔物を2体配置して子魔物を生産 ⚠️未確定 |

## 5. ペット（同行）システム

### 5-1. 概要

救出した霊魂または捕獲した魔物を1体選び、**ダンジョン探索に同行させる**ことができる。

- 同行できるのは**1体のみ**（将来的な上限解放は施設強化で検討）。
- ペットはプレイヤーの後ろのマスをついてくる（追従AI）。

### 5-2. ペットの追従 AI

```
プレイヤーが移動 → ペットはプレイヤーの「1ターン前の位置」に移動する
（列車の連結のような挙動）
```

| 状況 | 動作 |
|---|---|
| プレイヤーが連続移動 | プレイヤーの直後を追従 |
| 狭い通路 | プレイヤーの後ろのマスに並ぶ（通路を塞がない） |
| プレイヤーから2マス以上離れた | 自動でプレイヤーへ向かって A* 移動 |
| プレイヤーがワープ（落とし穴等） | ペットは次のフロアの出発地点に自動転送 |

### 5-3. ペットの戦闘 AI

| ステート | 条件 | 行動 |
|---|---|---|
| **待機** | プレイヤーの周囲3マスに敵がいない | ただ追従する |
| **援護攻撃** | 隣接マスに敵がいる | 1ターンに1回通常攻撃を行う |
| **逃走補助** | ペットのHP が 20% 以下 | プレイヤーのそばから離れず、攻撃を停止 |

### 5-4. ペット固有スキル（魔物タイプ別）

| ペットタイプ | 固有スキル名 | 効果 |
|---|---|---|
| スライム系 | 吸収 | 倒した敵からランダムアイテムを1つ自動回収 |
| 鳥類系 | 偵察 | 周囲5マスの霧を解除する（ミニマップに映す） |
| ゴーレム系 | 壁立て | 1ターン、プレイヤーへの攻撃を身代わりに受ける |
| 草食魔物 | 草むしり | 採取ポイントからの素材獲得量 +1 |
| アンデッド系 | 呪縛 | 攻撃した敵を1ターン行動不能にする（2ターンに1回） |

### 5-5. ペットの死亡処理

- ペットのHPが0になると**「消滅演出」**が流れ、そのダンジョン内では復活しない。
- ダンジョンを生きて脱出した場合、**ペットは拠点で「気絶状態」として復活する**。
- プレイヤーが死亡して帰還した場合も、ペットは**拠点に自動帰還する**（ロストしない）。

## 6. 拠点 NPC との会話・親密度システム

### 6-1. 町の NPC との会話

- 拠点に配置された霊魂には話しかけることができる。
- 会話内容は**親密度の段階**に応じて変化する（3〜4段階）。
- 親密度が一定値を超えると**「特技解放」や「ボーナス強化」**が起きる。

### 6-2. 親密度の上昇条件（案）

| 行動 | 上昇量 |
|---|---|
| 話しかける（1日1回） | +5 |
| 霊魂の好む素材・アイテムをプレゼント | +10〜30（アイテムによる） |
| 特定クエストをクリアする | +50（イベント限定） |
| 施設にずっと配置し続ける | +1/日（自動） |

### 6-3. 親密度段階（案）

| 段階 | 親密度値 | 解放内容 |
|---|---|---|
| 知人 | 0〜49 | デフォルトの会話 |
| 友人 | 50〜149 | 施設ボーナス+5%強化、新会話追加 |
| 親友 | 150〜299 | 特技強化、固有クエスト解放 |
| 絆 | 300〜 | 特技Lvアップ、特別衣装（依り代の見た目変化） |
