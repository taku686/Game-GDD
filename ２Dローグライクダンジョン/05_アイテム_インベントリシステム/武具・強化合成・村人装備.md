# 武具・強化合成・村人装備

本作の装備品（武器・防具・アクセサリー）は、冒険者が装備してダンジョンで戦うためのステータス向上効果に加えて、**「町の施設にいる住人（霊魂）の依り代に持たせる（装備させる）ことで、生活職・ミニゲームの効率を上げる仕事道具となる」**という非常に重要な役割を持ちます。

## 装備スロットの仕様：冒険者と村人の違い

本作では「プレイヤー（冒険者）」と「村人（霊魂）」で武器枠の仕様が異なります。

*   **冒険者の武器枠（２スロット）：**
    右手・左手の概念があり、**武器用スロットを最大２つ**まで持っています。
    *   **片手武器：** スロットを１つ消費。左右の手に１つずつ装備可能（二刀流、または片手武器＋盾など柔軟な組み合わせが可能）。
    *   **両手武器：** スロットを２つ消費。両手を使って大きく振るう必要があり、これを装備すると他の武器や盾は持てません（強制で１つのみの装備となる）。
*   **村人の武器枠（１スロット固定）：**
    村人（依り代）は拠点での特定の作業に特化しているため、**武器の両手・片手を問わず一律で「仕事道具として１つだけ」**装備させることができます。

## 武器種一覧と「村人（生活職）への適性」
※武器ごとに「片手 / 両手」の区分があり、冒険者のプレイスタイルに大きく影響します。

| 武器の種類 | 区分 | 【ダンジョン】冒険者の主な戦闘スタイル（ターン制ローグライク） | 【拠点】住人に装備させた際の効果・適性 |
| :--- | :--- | :--- | :--- |
| **剣（片手剣）** | 片手 | 正面1マスへの標準的な攻撃。命中・火力のバランスが良く盾と合わせやすい。 | **護衛・運び屋**：汎用戦闘武器として行商人に持たせ、道中のアイテムロスト率低下や護衛役として機能。 |
| **大剣 / 太刀** | 両手 | **前方3方向**（正面と斜め）を同時攻撃できるが、命中率がやや低い。 | （同上。重武装な護衛役として機能） |
| **ナイフ / 短剣** | 片手 | 基本攻撃力は低いが、稀に**1ターンに2回攻撃**が発生したり、会心が出やすい。 | **料理人・解体業**：食材からの料理ボーナスや、ガラクタ解体時のレア素材獲得率UP。 |
| **斧** | 片手 | 攻撃が外れやすい代わりに、会心率が高く、敵の防御バフを無視する。 | **木こり（資材調達）**：建材となる「木材」のパッシブ収集量の増加、レア木材の発見率アップ。 |
| **ハンマー** | 両手 | 攻撃時に高確率で敵を「**ノックバック（後退）**」させたり「気絶（数ターン行動不能）」させる。 | **鍛冶屋（強化・建築）**：武器の強化成功率アップや、施設を拡張する際の時間短縮・コスト削減。 |
| **鎌** | 両手 | 自分の**周囲8マス（全方位）**を一斉に攻撃できる。モンスターハウス等で真価を発揮。 | **農家（農業・採取）**：畑の農作物の収穫量アップや、成長速度の上昇。 |
| **槍 / ランス** | 両手 | **前方2マス**まで届く貫通攻撃。敵に隣接されずに先制攻撃でき、2体まとめて刺せる。 | **漁師（釣り）**：釣り用ミニゲームで「銛（もり）」として機能。レアな大物や深海魚のヒット率・獲得品質が上がる。 |
| **杖 / 魔導書** | 片手 | 振ると直線上（部屋の奥まで）に魔法効果（ダメージ・状態異常など）を放つ。 | **鑑定士・学者**：未鑑定アイテムのレアリティ上昇、鑑定速度のアップ。魔法施設の研究ボーナス。 |
| **弓** | 両手 | 矢（アイテム）を消費し、直線上遠くの敵を安全に削る。引き寄せやワープ矢などの絡め手にも。 | **狩人（狩猟）**：特定の肉や毛皮などのドロップ素材をパッシブで収集する効率がアップ。 |
| **爪 / ナックル** | 片手 | 攻撃時に敵のシールドを一定確率で破壊する、あるいは必中攻撃になりやすい。 | **魔物使い（ブリーダー）**：モンスターの育成速度アップ（手入れ用具）や、テイムした魔物のステータスボーナス。 |
| **扇 / 楽器** | 片手 | 直接攻撃より、振ることで**部屋全体**の敵を高確率で状態異常（睡眠や混乱など）にする。 | **酒場の店員**：「踊り子」や「給仕」として酒場客の初期スタミナを増やしたり、運（レア報酬率）を上げる。 |
| **盾** | 片手 | 防御力アップの基本。印によって「満腹度減少抑制」や「盗み防止」などの効果を発揮する。 | **街の警備兵**：装備させると、拠点施設がトラブルに見舞われるのを防ぐなど、村人全体を底上げする防御系パッシブ。 |

（※これによって、「自分は二刀流ビルドだから両手武器の大剣は装備できないけど、村人のレベル上げに使えるから当たりだ！」といった喜びが生まれます）

## 装備品のステータスと「適性」の連動について
武器には「攻撃力」「属性（火・水など）」「耐久度（またはレアリティ）」などのパラメータが存在します。

*   **基礎ステータスの影響：** 同じ「ハンマー」でも攻撃力が高いほど鍛冶の「大成功率」が上がったり、ミニゲームの難易度が下がったりします。
*   **属性の影響（重要）：**
    *   `火属性のハンマー`: 鍛冶屋が使うと、強化した武器に火属性エンチャントが付きやすくなる。
    *   `水属性の鎌`: 農家が使うと、畑に水が撒かれて作物の成長スピードが劇的に上がる。
*   **レアリティの影響：** 霊魂にはその器（鎧）との適性による「装備コスト制限」のようなものがあり、施設のレベル（または拠点の復興度）が上がらないと強い魔力を持つ最高レアリティの武器は装備させられない仕組み（＝拠点と装備のインフレを連動させる）を想定します。

## 防具（鎧・服）とアクセサリーの役割・設定

武器が「生活職に対するアクションの道具（特定スキル・効率ボーナス）」を提供するのに対し、防具（鎧や服）は霊魂が物理世界で活動するための**「より強固な依り代（器）」**として機能し、アクセサリーは**「作業に特定の補正をかける魔導装置」**として働きます。
これらも「武具生産の国」の技術で作られているため、単なる布切れや宝石ではなく、霊的なエネルギー（霊素）を流し込むことで特別な効果を発揮します。

### 防具（鎧・服）の種類と「村人（依り代）」への効果
防具は霊魂の「器の性質」を変化させ、拠点での基本的な作業効率（スタミナやスピード）を底上げします。

| 防具カテゴリ | 【ダンジョン】冒険者への効果 | 【拠点】住人（霊魂）に装備させた際の効果・器の性質 | 適した生活職の例 |
| :--- | :--- | :--- | :--- |
| **重鎧（プレートなど）** | 高防御・重量級 | **「スタミナ・耐久性アップ」**<br>器としての耐久度が高く、長時間働き続けても疲労しにくい（霊素の消費速度が低下する）。 | 鍛冶屋、木こり、警備兵 |
| **軽鎧（レザーなど）** | 防御と回避のバランス | **「作業スピードアップ」**<br>身軽な器となり、パッシブでのアイテム生産速度や、ミニゲーム時の基本アクション速度が上昇する。 | 農家、狩人、運び屋 |
| **魔法衣（ローブなど）** | 魔法耐性・魔力アップ | **「魔力・レアリティブースト」**<br>霊素との親和性が極めて高い器。装備させると、作業の大成功率やレアアイテムの発見率・獲得品質が上昇する。 | 鑑定士、学者、魔法施設 |
| **平服（町人服・給仕服など）** | 低防御・特殊効果 | **「モチベーション向上・特殊補正」**<br>生前の姿に近い人間らしい器。施設の客（プレイヤー）へ与えるバフ効果量が上がったり、固有の特殊イベントが起きやすくなる。 | 酒場の店員、料理人 |

### アクセサリーの種類と機能
王国特産の魔石や鉱石を使ったアクセサリーは、特定の作業に対する「局所的なバフ」や「ミニゲームの難易度緩和」の役割を担います。

*   **腕輪・指輪系（手元の補助）**
    *   **効果：ミニゲームの各種判定・難易度の緩和**（ゲージの減少速度低下、ヒットエリアの拡大など）。
    *   *例：「技巧の指輪」を漁師に装備させると、釣りミニゲームの成功判定が甘くなる。*
*   **首飾り・ペンダント系（運気・品質の向上）**
    *   **効果：特定の素材のドロップ率や、一度の獲得「量」に対するボーナス。**
    *   *例：「豊穣のネックレス」を農家に装備させると、収穫時に確率で農作物が２倍ドロップする。*
*   **護符・お守り系（トラブル回避・安定）**
    *   **効果：拠点での「失敗・事故」確率の低下、または状態異常への防御。**
    *   *例：「安全の護符」を鍛冶屋に装備させると、装備強化の失敗時のペナルティを軽減する。*

**【装備枠のエコシステム】**
村人（霊魂）の装備枠は、「武器（仕事道具）１枠 ＋ 防具（器）１枠 ＋ アクセサリー１枠」などを想定しています。
ダンジョン攻略で行き詰まった際や、上位の装備を手に入れた際に、**「プレイヤーの型落ち装備を村人に着せることで、拠点がどんどん便利になり、結果的にプレイヤー自身の強化・救済に繋がる」**というループを生み出します。
