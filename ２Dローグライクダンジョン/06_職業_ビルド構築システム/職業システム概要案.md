# 職業システム概要案（厳選版）

2Dローグライクにおける「ビルドの構築しやすさ」「各クラスのプレイスタイルの差別化」を重視し、スプレッドシートの案を厳選・再構築したシステムです。
基本職5種をベースにし、その「2種類の組み合わせ」で全10パターンの上級職が生まれる、非常に美しく分かりやすい構成になっています。

---

## 1. 基本職（5種類）
冒険の初期から選択できる、各ロール（役割）のスペシャリストです。

*   **戦士** 【近接アタッカー / タンク】
    *   高いHPと防御力で前線を支える盾役。剣や斧での近接戦闘が得意。
*   **魔法使い** 【遠距離魔法アタッカー】
    *   火・氷・雷などの属性魔法で遠距離から敵を殲滅する。MP管理が重要。
*   **僧侶** 【ヒーラー / バッファー】
    *   回復や蘇生、能力アップの魔法を操る。アンデッドに強い。
*   **盗賊** 【スピードアタッカー / 探索】
    *   素早い動きでアイテムを盗み、罠を解除する。短剣による手数の多さが特徴。
*   **狩人** 【遠距離物理アタッカー / 妨害】
    *   弓矢による遠距離攻撃と、罠による敵の足止めが得意。敵の弱点を見抜く。

---

## 2. 上級職（全10種類）
「基本職A」＋「基本職B」のレベルを上げるなどの条件を満たすことで解放される上位クラスです。基本の5職の「全ての組み合わせ（5×4÷2＝10通り）」に対応しています。

1.  **剣聖** 【戦士 ＋ 盗賊】（近接の極み）
    *   盾を捨てて双剣などを極めたクラス。圧倒的な攻撃速度と手数で敵を刻む。
2.  **パラディン** 【戦士 ＋ 僧侶】（究極の盾）
    *   自己回復や防御魔法を扱い、味方をかばう不沈艦。
3.  **狂戦士** 【戦士 ＋ 狩人】（背水のアタッカー）
    *   自らのHPを犠牲にして攻撃力を高める、ハイリスク・ハイリターンの物理職。
4.  **魔法戦士** 【戦士 ＋ 魔法使い】（弱点特効の剣士）
    *   自身の近接武器に属性魔法をエンチャント（付与）し、敵の弱点を的確に突く。
5.  **錬金術師** 【魔法使い ＋ 盗賊】（アイテム・妨害特化）
    *   ポーションでの支援や爆弾アイテムでの攻撃、装備の即席強化などトリッキーな戦い方をする。
6.  **賢者** 【魔法使い ＋ 僧侶】（万能魔法使い）
    *   攻撃魔法と回復魔法の両方を極め、広範囲の敵を一掃する。
7.  **自然魔導士** 【魔法使い ＋ 狩人】（環境操作魔法）
    *   植物や岩など「地形」に影響を与える魔法を操り、敵を足止め・妨害する。
8.  **聖なる射手** 【僧侶 ＋ 狩人】（支援特化スナイパー）
    *   後方から安全に味方を回復しつつ、浄化の矢でアンデッドを射抜くサポートアタッカー。
9.  **吟遊詩人** 【僧侶 ＋ 盗賊】（全体バフ・デバフ）
    *   歌や楽器を使って、広範囲の味方にバフ（強化）、敵にデバフ（弱体化）をばら撒く。
10. **探求者** 【狩人 ＋ 盗賊】（探索・ユーティリティの極み）
    *   隠し部屋の発見、罠の完全無効化、未鑑定アイテムの判別など、ダンジョン探索において右に出る者はいない。

---

## 3. マスター職（隠し要素）
やり込み要素として、極めて厳しい条件（例：すべての基本職を最大レベルにする 等）で解放れる特別なクラス。バランスブレイカーとなるため、基本は1職のみとします。

*   **勇者** 【全ての力を結集した者】
    *   物理、魔法、回復、探索すべてにおいて最強の恩恵を受ける伝説の存在。

---

## 4. 生活職（拠点・ミニゲーム用サブジョブ）
ダンジョン探索の戦闘とは切り離し、システム『[09_ミニゲームシステム]』や町育成に特化したサブスキル的扱いの職業です。
それぞれ、拠点にある特定の「施設」と強く連動して恩恵をもたらします。プレイヤーのサブジョブとしてセットしたり、その職業に適正を持つ霊魂（住人）を拠点に配置することで効果を発揮します。

*   **釣り師**【連動施設：釣り堀】
    *   釣りミニゲームの難易度低下（制限時間延長やシールド追加）。拠点「釣り堀」での食材自動獲得量UPやレア魚の確率UP。
*   **農民 / 採集家**【連動施設：畑（農場）】
    *   落下物キャッチ（収穫）ミニゲームの難易度低下。畑の収穫量アップ・種からの成長日数短縮。
*   **料理人**【連動施設：料理屋（食堂）】
    *   料理・体内時計ミニゲームの難易度低下。調理済み食事アイテムの「永続バフ効果」の付与や一時バフの効果時間延長。
*   **鍛冶職人**【連動施設：鍛冶屋】
    *   連打ミニゲーム等の緩和。「鍛冶屋」での武器修復・強化にかかる費用の割引、および強化大成功率UP。
*   **魔物使い / ブリーダー**【連動施設：牧場】※新規追加
    *   ダンジョン内での「モンスター捕獲アイテム（ネット等）」の成功率UP。拠点「牧場」で飼育しているモンスターからの「卵・ミルク」等の生産量増加。
*   **大工 / 建築家**【連動施設：役場・建築所】※新規追加
    *   木を伐採するミニゲームの緩和。町全体の「新たな施設の建設・アップグレード」に必要となる建築素材や資金を割引（軽減）する町の要。
*   **商人 / 運び屋**【連動施設：各種ショップ・倉庫】※新規追加
    *   「武器屋・防具屋・道具屋」でのアイテム購入時の割引・売却額UP。また、ダンジョン内での所持枠（インベントリスロット）の一時的な拡張。
*   **鑑定士 / 学者**【連動施設：道具屋・酒場】※新規追加
    *   未鑑定アイテム（ポーションや巻物）をノーコストで鑑定できるようになる。また、酒場でのクエスト報酬金が割り増しされる。
