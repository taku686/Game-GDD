# スキルシステム概要

## 基本方針

スキルは「職業レベル」「主人公スキルツリー」「施設レベル」の3層構造で習得・強化される。

- **主人公の霊魂**：職業Lv（Layer 1）＋ スキルツリー（Layer 2）の2軸で成長する
- **住民の霊魂（NPC）**：スキルツリーは持たない。**レア度**によって個体の強さ・特性・施設効率が決まる

---

## 3層構造の全体像

```
Layer 1：職業スキル（冒険者職業）
  └ 各職業に独立したLvがあり、Lvアップで固有スキルを習得（DQ方式）
  └ 主人公・住民の霊魂ともに対象

Layer 2：主人公スキルツリー（主人公専用）
  └ ゲーム全体の活動で経験値を獲得し、分岐型ツリーを解放
  └ ステータス・耐性・スキルスロット・職業横断パッシブを強化
  └ 住民の霊魂はツリーを持たず、レア度で差別化

Layer 3：生活職業スキル（施設連動）
  └ 特定の施設レベルアップ、または生活職業のLvアップでスキルを習得・強化
  └ 探索補助・生産効率・町運営に特化した非戦闘スキル群
```

---

## Layer 1：職業スキル

### 概要
- 職業ごとに独立した「職業Lv」が存在する（DQの「職業熟練度」方式）
- 職業Lvは、その職業でダンジョンを探索することで上昇する
- Lvアップ時に対応スキルが自動習得される（スキルポイント制ではなく自動付与）
- 職業を変更しても、習得済みスキルは「習得済みスキルリスト」に保持される
  - ただし、使用できるのは**現在セットしている職業のスキル**のみ
  - 上級職転職後は、元の2つの基本職スキルも一部引き継いで使用できる

### 職業Lvの上限と習得スキル数（案）

| 職業種別 | Lv上限 | 習得スキル数 |
|---|---|---|
| 基本職（戦士・魔法使い・僧侶・盗賊・狩人） | Lv10 | 各職業 10個程度 |
| 上級職（全10種） | Lv15 | 各職業 15個程度（引継ぎ含む） |
| マスター職（勇者） | Lv20 | 全職業の精鋭スキル群 |

> ※ スキル数・Lv上限は今後の調整項目。

### スキルの種別

| 種別 | 内容 | 例 |
|---|---|---|
| **アクティブスキル** | 戦闘中に任意で発動する技・魔法。MPや霊素を消費する | 「斬撃」「ファイアボール」「回復魔法」 |
| **パッシブスキル** | 常時発動する能力強化。発動コストなし | 「HP+10%」「会心率+5%」「移動コスト-1」 |
| **戦術スキル** | 特定条件下で発動する条件付き効果 | 「瀕死時に攻撃力2倍」「同一敵に3回攻撃後、次の攻撃が必中」 |

### スキルスロット制限
- 戦闘中に使用できるアクティブスキルの数は「スロット」で管理
- 初期スロット数：4スロット（パッシブは常時有効でスロット不要）
- スキルツリー（Layer 2）や特定施設のLvアップでスロット数が増加する

## Layer 2：主人公スキルツリー（主人公専用）

### 概要
- **主人公の霊魂のみ**が持つ成長システム
- ゲーム内のあらゆる活動で「経験値」を獲得し、スキルツリーのノードを解放する
- ツリーは**分岐型**：途中で進む方向を選ぶことで、プレイスタイルに合わせた特化が可能
- ツリーで強化できる内容はステータス・耐性・スキルスロット・職業横断パッシブなど

> **住民の霊魂（NPC）との差別化**：住民霊魂はスキルツリーを持たない。
> 救出時に決まる「レア度」が施設効率・固有特性・装備適性の強さを決める。
> （住民のレア度仕様については `07_キャラクター管理システム` に別途記載予定）

### 経験値の獲得源

| 活動 | 獲得できる経験値 |
|---|---|
| ダンジョン戦闘（敵撃破） | 多め |
| ダンジョン探索（フロア踏破・宝箱発見） | 中程度 |
| 帰還（ダンジョンから生還） | 中程度 |
| 料理・食事 | 少量 |
| ミニゲームクリア（釣り・収穫など） | 少量 |
| クエスト達成 | 多め（ボーナス）|

> どの活動でも経験値が入るため、「今日はのんびり農業」「今日はダンジョン集中」どちらのプレイでも主人公が育つ。

### ツリーの構造（分岐型）

```
        ［スタート：初期ノード群］
              ↓
    ┌─────────┼─────────┐
  ［攻撃特化ブランチ］  ［耐久特化ブランチ］  ［探索特化ブランチ］
   ・攻撃力強化          ・HP・防御強化        ・移動コスト削減
   ・会心率アップ         ・属性耐性アップ       ・スキルスロット拡張
   ・スキルダメージ倍率    ・状態異常耐性         ・アイテム発見率UP
```

> ブランチは途中から投資先を変えることも可能だが、深い強化は一方向に集中した方が効果が高い。

### ツリーで強化できる内容

> ※ ツリーで強化できる内容の詳細は後日設計予定。

### 職業Lv × スキルツリー の組み合わせ例

| 構成 | 特徴 |
|---|---|
| 職業Lv高・ツリー未発達 | 強力な職業スキルを持つがステータスが低く、スロット数も少ない |
| 職業Lv低・ツリー発達 | 耐久・耐性が高くスロットも多いが、使えるスキルは基礎的なものが中心 |
| 両方発達（ゲーム終盤） | 高ステータス＋多彩なスキル構成で強力なビルドを実現できる理想状態 |

## Layer 3：生活職業スキル（施設連動）

### 概要
- 生活職業（釣り師・農民・料理人・鍛冶職人など）にも独立したLvが存在する
- 生活職業Lvは、対応する施設での活動・ミニゲームのクリアで上昇する
- 施設のLvアップと生活職業Lvアップの両方でスキルが強化される

### 生活職業スキルの特徴
- 基本的に非戦闘スキル（ダンジョン探索補助・生産効率UP・町運営に関わる効果）
- 霊魂（主人公・住民ともに）を施設に配置する場合、その霊魂の生活職業Lvが施設効率に影響する
  - 住民霊魂は生活職業Lvの代わりに**レア度**が施設効率の主な決定要因になる
- プレイヤー自身がセットするサブジョブとしても機能する

### 生活職業スキル一覧（主要）

| 生活職業 | 連動施設 | スキル例（Lv上昇で強化） |
|---|---|---|
| 釣り師 | 釣り堀 | ミニゲーム補助、レア魚出現率UP、食材自動獲得量UP |
| 農民・採集家 | 畑 | 収穫量UP、作物成長速度UP、ミニゲーム補助 |
| 料理人 | 料理屋 | バフ料理の効果時間延長、永続バフ付与、ミニゲーム補助 |
| 鍛冶職人 | 鍛冶屋・武器屋 | 強化費用割引、大成功率UP、製造速度UP |
| 防具職人 | 防具屋 | 防具・盾・アクセサリーの製造速度UP、クオリティ底上げ |
| 魔物使い | 牧場 | モンスター捕獲成功率UP、生産量UP |
| 大工・建築家 | 役場 | 施設建設コスト割引、建設速度UP |
| 商人・運び屋 | 道具屋・各店舗 | 購入割引、売却額UP、インベントリ一時拡張 |
| 鑑定士・学者 | 鑑定所・酒場 | 未鑑定アイテムをノーコスト鑑定、クエスト報酬割増 |

> 詳細は `職業システム概要案.md` の「生活職」セクションも参照。

## スキル習得の全体フロー

```
【主人公】
ゲーム内の全活動（戦闘・生活・探索・食事など）
  ↓（経験値獲得）
スキルツリーのノードを解放
  └ ステータス・耐性・スロット拡張・職業横断パッシブを自由に強化

その職業でダンジョンを探索
  ↓（職業Lv UP）
職業固有スキルを自動習得（アクティブ・パッシブ・戦術）

施設での活動（ミニゲーム・製造など）
  ↓（生活職業Lv UP）
生活職業スキルが強化・解放

【住民の霊魂（NPC）】
救出時のレア度で施設効率・固有特性が決まる（成長なし）
  └ 装備させることで施設効率をさらに底上げ可能
```
