# 武器屋（武器工房）

## 概要
武器の「設計図」と「素材」を消費して武器を製造する生産工房。購入ではなく「製造」が主体の施設であり、リアルタイムで時間が経過した後に武器が完成する。完成した武器は探索への持ち込みや村人への装備に使用できる。

製造だけでなく、手持ちの武器の**強化・ルーン（特殊効果）付与**も行える。鍛冶素材を消費してミニゲームに挑み、操作スコアに応じて強化の成功度やボーナスが変動する。

武器の販売（既製品の購入）は**道具屋**の役割。武器屋はあくまで「作る場所・育てる場所」として機能する。

## 基本情報

| 項目 | 内容 |
|---|---|
| **カテゴリ** | 探索準備・装備 |
| **建設タイプ** | 複数建設可能（最大3件） |
| **アンロック条件** | 武器職人の霊魂をダンジョンで救出後にアンロック |

## 役割

### 武器の製造フロー
1. **設計図の選択**：アンロック済みの設計図から作りたい武器を選ぶ（設計図はダンジョン・クエスト・行商人で入手）
2. **素材の投入**：その武器に必要な素材（鉱石・木材・魔法の粉など）を消費する
3. **生産開始**：製造キューに追加される。リアルタイムで時間が経過し、完成するまで待つ
4. **受け取り**：完成後に武器を受け取り、インベントリまたは保管庫へ格納

### 複数建設の効果
武器屋を複数建設すると**並列で製造できる数**が増加する。例えば3件建てれば3つの武器を同時に製造できる。

### 生産時間の目安

| 武器レアリティ | 目安の生産時間 |
|---|---|
| コモン | 5〜15分 |
| アンコモン | 30分〜1時間 |
| レア | 3〜6時間 |
| エピック | 12〜24時間 |
| クリア後（魔界武器） | 2〜3日 |

> ※ 配置した霊魂の能力・装備・施設レベル・時間短縮アイテムによって大幅に変動する。

---

### 武器の強化・ルーン付与

製造した武器（または既存の武器）を素材を消費してさらに育てる。強化の際はミニゲームが発生し、操作スコアに応じて成功度や付与ボーナスが変動する。

| フロー | 内容 |
|---|---|
| ① 対象選択 | 強化したい武器を選ぶ |
| ② 種別選択 | 「通常強化（攻撃力・会心率などの数値アップ）」か「ルーン付与（特殊効果の付与）」かを選ぶ |
| ③ 素材消費 | 鉱石・魔法の粉などを消費する |
| ④ ミニゲーム | 操作スコアに応じて強化の成功度・大成功ボーナスが変動する |

施設レベルが上がると扱える強化ランクとルーンの種類が増加し、より強力な武器へと育てられる。

## 霊魂配置

> 霊魂の配置・変更はこの施設で直接行うか、**役場**の一括管理UIからも行うことができる。受け入れ可能なスロット数は施設レベルによって決まる。

| 項目 | 内容 |
|---|---|
| **配置スロット数** | Lv1：1体 → Lv3：2体 → Lv5(MAX)：3体 |
| **最適な武器種** | ハンマー（両手） |
| **武器の効果** | 生産速度アップ・武器の品質クオリティが上昇しやすくなる |
| **最適な防具** | 重鎧（プレートなど） |
| **防具の効果** | 同時製造キューの上限が増加する |
| **属性ボーナス** | 火属性のハンマー：完成した武器に火属性エンチャントが自動付与される確率アップ |
| **霊魂レベルの影響** | 霊魂のレベルが高いほど S・SS クオリティが出やすくなる（レベルと装備の両方が影響） |

> **差別化のポイント**：同じ設計図・素材を使っても、配置した霊魂のレベルと装備によって完成品のクオリティが大きく変わる。より良い武器を作りたければ、強い霊魂を救出して良い装備を持たせる必要がある。

## レベルアップ効果

| レベル | 解放内容・強化内容 |
|---|---|
| Lv1 | コモン武器の製造が可能。基本強化（攻撃力の数値アップ）が可能。製造キュー数：1（並列1件まで） |
| Lv2 | アンコモン武器の製造が解放。全体の生産速度が上昇。コモンルーンの付与が解放 |
| Lv3 | レア武器の製造が解放。霊魂配置スロット+1。レアルーンの付与が解放 |
| Lv4 | エピック武器の製造が解放。製造キュー数増加（並列2件まで）。強化失敗時のペナルティ軽減。エピックルーンの付与が解放 |
| Lv5（MAX） | 全レアリティの武器製造が可能。品質上昇の確率が全体的にアップ。最高レアルーンの付与解放・複数ルーン同時付与可能。改造オプション解放 |

## 武器クオリティシステム

完成した武器には C〜SS の5段階のクオリティランクが付く。クオリティはステータス値の高さ・エンチャントスロット数・特殊特性の有無に影響する。

### クオリティランク一覧

| ランク | 目安の出現率（通常工房・Lv1） | 特徴 |
|---|---|---|
| C | 15% | 最低品質。最低限使えるが強化余地は少ない |
| B | 40% | 標準品質。序盤〜中盤の主力 |
| A | 30% | 高品質。エンチャントスロット+1 |
| S | 12% | 優品質。特殊特性が稀に付与される |
| SS | 3% | 傑作品。特殊特性が確定付与される |

> S・SS ランクの「特殊特性」は基本ステータスに上乗せされる追加効果（例：会心ダメージ+25%、攻撃時に炎上付与、霊素微回復 など）。

### クオリティを決める4つの要素

| 要素 | 内容 |
|---|---|
| **① 霊魂のレベルと装備** | 配置霊魂のレベルが高く、高レアリティのハンマーを持たせるほど S・SS を引く確率が上昇（最重要） |
| **② 施設レベル** | レベルが上がるほど C ランクの出現率が下がり、S+SS 確率にボーナスが加算される |
| **③ 改造オプション** | 「品質特化工房」を選ぶと C・B が完全排除され、最低 A ランク保証になる |
| **④ 課金アイテム** | SS クオリティ確定（職人の魂結晶）や1ランク底上げ（品質の秘薬）が可能 |

---

## 改造オプション（Lv5 MAX後に解放）

> いずれか1つを選択。選択後は施設を解体しない限り変更不可。
> 複数建設している場合は施設ごとに別の改造オプションを選べる。

### 量産工房
生産速度が大幅アップし、製造キューの上限も増加する。クオリティの確率分布は C・B 寄りになるが、とにかく数をそろえたい運用スタイル向け。

### 品質特化工房
C・B ランクが出現しなくなり、最低 A ランク保証になる。同じ設計図・素材でも格段に良い武器が作れる。こだわりの一本を育てるスタイル向け。生産速度はやや低下。

### 研究開発工房
新しい設計図を「開発（発見）」する機能が追加される。素材とゴールドを消費して一定時間研究すると、ダンジョン入手に依存しない設計図を自力で生成できる。

### 強化特化工房
製造よりも強化・ルーン付与に特化した工房。強化時の攻撃系ステータスの強化値が×1.5倍になり、ルーン付与枠が+1される。ルーン付与の大成功率もアップ。製造機能は維持されるが生産速度はやや低下する。エンチャントビルドを深く追いたいプレイヤーに最適。

## 課金アイテムと無課金手段

### 通常手段（無課金）
| 手段 | 効果 |
|---|---|
| 霊魂のレベルアップ | S・SS の出現確率アップ（最重要） |
| 霊魂にハンマー装備 | レアリティに応じて S+SS 確率アップ |
| 施設レベルアップ | C ランク排除・S+SS 確率ボーナス加算 |
| 「品質特化工房」への改造 | C・B 完全排除、最低 A 保証 |
| 「量産工房」への改造 | 生産速度大幅アップ |
| 料理屋のバフ料理 | 一時的な生産速度アップ（探索1回分） |

### 課金アイテム（有料）
| アイテム名（仮） | 効果 | 種別 |
|---|---|---|
| **職人の魂結晶** | 次の製造を SS クオリティ確定にする | クオリティ保証 |
| **品質の秘薬** | 次の製造のクオリティを1ランクアップ | クオリティ底上げ |
| **鍛冶の加速石** | 指定した製造中の武器を即時完成させる（クオリティに影響なし） | 時間短縮 |
| **工房の神炎** | 30分間、すべての製造速度を10倍にする | 時間短縮 |

> **設計上の注意**：課金要素は「クオリティ保証」と「時間短縮」に限定し、課金しないと入手できない武器・設計図は作らない。無課金でも全要素を揃えれば SS が25%前後になる設計を前提とする。

## 備考
- 武器の**販売（既製品の購入）は道具屋**が担当。道具屋では基本的な武器の既製品を購入できる（設計図不要・即時入手）。武器屋は「こだわりの武器を作り・育てる」場所として差別化される。
- 設計図の入手方法：ダンジョンの宝箱・ボスドロップ・クエスト報酬・行商人・研究開発工房（改造後）。
- 強化ミニゲームの詳細仕様は `09_ミニゲームシステム/武器・防具屋の強化ゲーム.md` で管理。
- 強化に必要な素材（鉱石・魔法の粉など）は `05_アイテム_インベントリシステム/資源・クラフト素材仕様.md` を参照。
