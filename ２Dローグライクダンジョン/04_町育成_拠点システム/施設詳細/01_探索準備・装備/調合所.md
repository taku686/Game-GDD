# 調合所（錬金術アトリエ）

## 概要
薬草・モンスタードロップ素材などを使って戦闘用消耗品を作成・合成する施設。道具屋での購入とは異なり、素材を持ち込むことで高性能アイテムを自作できる。

## 基本情報

| 項目 | 内容 |
|---|---|
| **カテゴリ** | 探索準備・装備 |
| **建設タイプ** | 1施設1つ固定（レベルアップ式） |
| **アンロック条件** | 錬金術師の霊魂をダンジョンで救出後にアンロック |

## 役割
「強力な回復ポーション」「爆弾」「状態異常回復薬」「属性付与の薬」などを素材から作成する。また、下位素材を上位素材に変換する「精製機能」も持ち、素材管理の中核を担う。

道具屋が「手軽さ」を担う施設であるのに対し、調合所は「素材を使って高品質なものを自作する」という深みのある強化手段を提供する。

## 霊魂配置

> 霊魂の配置・変更はこの施設で直接行うか、**役場**の一括管理UIからも行うことができる。受け入れ可能なスロット数は施設レベルによって決まる。

| 項目 | 内容 |
|---|---|
| **配置スロット数** | Lv1：1体 → Lv3：2体 → Lv5(MAX)：3体 |
| **最適な武器種** | 杖/魔導書（片手） |
| **武器の効果** | 調合時の成功率アップ・作成物に魔法ボーナス効果が付く確率アップ |
| **最適な防具** | 魔法衣（ローブなど） |
| **防具の効果** | 霊素との親和性が高まり、作成した消耗品のレアリティと大成功（特上品生成）率がアップ |
| **属性ボーナス** | 火属性の杖：爆弾・炎系アイテムの性能が大幅アップ。水属性の杖：回復ポーションの回復量アップ |
| **レアリティの影響** | 高レアリティの杖を持たせるほど、調合で作れる消耗品の品質上限が上がる |

## レベルアップ効果

| レベル | 解放内容・強化内容 |
|---|---|
| Lv1 | 基本的なポーション・簡易爆弾の作成が可能 |
| Lv2 | 中〜上位ポーションのレシピ解放。状態異常系アイテムの作成が可能に |
| Lv3 | 素材変換機能（下位→上位）が解放。霊魂配置スロット+1 |
| Lv4 | 属性付与の薬・特殊ボム系レシピが解放 |
| Lv5（MAX） | 最高品質アイテムのレシピ解放。複数アイテムの同時調合が可能に。改造オプション解放 |

## 改造オプション（Lv5 MAX後に解放）

> いずれか1つを選択。選択後は施設を解体しない限り変更不可。

### 爆発物工房
爆弾・炎瓶・毒ガス爆弾などの攻撃系消耗品の性能と作成効率が大幅アップ。爆弾系は威力×1.5倍・製造コスト削減。回復系アイテムの作成効率はやや低下する。

### 薬師工房
回復系・バフ系ポーションの性能アップに特化。回復量×1.5倍・バフ効果時間延長。また素材変換の変換率が向上し、下位素材から得られる上位素材の量が増加する。攻撃系アイテムの作成効率はやや低下する。

### 毒素研究所
状態異常系アイテム（毒・麻痺・睡眠・混乱）の品質が大幅アップ。複数の状態異常を同時に付与できる複合ポーションの作成が解放される。回復・爆発物系のコストはやや増加する。

## 備考
- 調合に使う薬草・素材の入手先は主にダンジョンの採取ポイントと牧場・畑。
- 素材変換機能は `04_町育成_拠点システム/拠点の資源管理.md` の建築素材管理と連動する。
