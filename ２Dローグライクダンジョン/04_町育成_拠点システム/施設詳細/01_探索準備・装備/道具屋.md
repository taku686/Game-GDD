# 道具屋（万能商店）

## 概要
ポーション（回復薬）・帰還アイテム・松明などの消耗品に加え、**武器・防具の既製品（基本品）**も販売する万能商店。「すぐに使える完成品をゴールドで買う」ことができる拠点唯一の総合店舗。

武器工房（武器屋）が「設計図+素材+時間をかけてこだわりの武器を製造する」施設であるのに対し、道具屋は「即座に購入できる手軽さ」が最大の強み。

## 基本情報

| 項目 | 内容 |
|---|---|
| **カテゴリ** | 探索準備・装備 |
| **建設タイプ** | 1施設1つ固定（レベルアップ式） |
| **アンロック条件** | ゲーム開始時から建設可能 |

## 役割
探索に必要なあらゆるアイテムを「ゴールドで即時購入」できる総合商店。以下の2つの販売カテゴリがある。

### カテゴリ①：消耗品
回復薬・帰還アイテム・松明・巻物・罠など、探索中の生存を支えるアイテムを販売。持ち込み数には上限があるため、何をどれだけ買うかの判断が重要。

### カテゴリ②：武器・防具の既製品
設計図不要・即時入手できる「基本品（コモン〜アンコモンランク）」の武器・防具を販売する。品揃えはランダムで定期的に更新される。

> **武器屋（工房）との使い分け**
> - 今すぐ武器が欲しい → 道具屋で購入（ゴールド消費・即時入手・品質は標準的）
> - 良い武器が欲しい → 武器屋で製造（素材消費・時間がかかる・品質は高い）

## 霊魂配置

> 霊魂の配置・変更はこの施設で直接行うか、**役場**の一括管理UIからも行うことができる。受け入れ可能なスロット数は施設レベルによって決まる。

| 項目 | 内容 |
|---|---|
| **配置スロット数** | Lv1：1体 → Lv3：2体 |
| **最適な武器種** | ナイフ/短剣（片手） |
| **武器の効果** | 刃物の扱いに慣れた店員を配置することで、入荷するアイテムの検品精度が上がり、通常のランダム入荷に加えて稀に「特品（品質の高い消耗品や装備品）」が在庫に追加される |
| **最適な防具** | 平服（町人服・給仕服など） |
| **防具の効果** | 人間らしい外見により顧客の安心感が上がり、在庫補充の頻度がアップする |
| **属性ボーナス** | 風属性のナイフ：入荷するアイテムに稀にコモンルーンが無料で付与される。在庫更新時に追加の希少アイテムが見つかる確率がアップ |
| **レアリティの影響** | 高レアリティのナイフを持たせるほど、稀に入荷するレアアイテム（特殊な巻物など）の品質が上がり、特品の入荷頻度も上昇する |

## レベルアップ効果

| レベル | 解放内容・強化内容 |
|---|---|
| Lv1 | 基本的な消耗品（小回復薬・松明・帰還アイテム）＋コモン武器・防具を販売 |
| Lv2 | 中回復薬・状態異常回復薬が入荷。アンコモン武器・防具が入荷。在庫数増加 |
| Lv3 | 大回復薬が入荷。特殊巻物の取り扱い開始。レア武器・防具が稀に入荷。霊魂配置スロット+1 |
| Lv4 | 上位消耗品（爆弾・特殊罠など）が入荷。エンチャント付き武器・防具が入荷 |
| Lv5（MAX） | 最上位回復薬・希少な巻物が入荷。エピック武器・防具が稀に入荷。改造オプション解放 |

## 改造オプション（Lv5 MAX後に解放）

> いずれか1つを選択。選択後は施設を解体しない限り変更不可。

### 希少品店
通常の消耗品販売に加え、ダンジョンでしか入手困難なレア素材・特殊素材をゴールドで直接購入できるようになる。鍛冶・合成に必要な希少鉱石や魔法素材をお金で補えるため、素材集めの手間を大幅に省ける。品揃えはランダムで帰還ごとに更新される。

### 格安商店
すべての商品が一定割合（-30%）で割引購入できる。レア素材の取り扱いはないが、回復薬・消耗品を安定して安く買い続けられるため、消耗品を大量に持ち込むスタイルのコストを大幅に削減できる。

## 備考
- 調合所（施設）で自作したポーションと消耗品販売の役割が重複するが、道具屋は「即座に購入できる手軽さ」が強み。調合所は「素材を使って高性能品を自作する」という差別化がある。
- 武器工房（武器屋）で製造する武器と武器既製品販売の役割が重複するが、道具屋は「今すぐ・ゴールドで・設計図不要」で購入できる手軽さが強み。武器屋は「素材をかけて品質の高い武器を作る」という差別化がある。
- 施設から購入できるアイテムの種類・在庫数はゲームバランス上重要な調整ポイント。更新タイミングは「ダンジョンから帰還するたびに一部ランダム更新」を想定。
