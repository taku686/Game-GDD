# 防具屋（防具工房）

## 概要
「設計図」と「素材」を消費して**防具・盾・アクセサリー**を製造する生産工房。購入ではなく「製造」が主体の施設であり、リアルタイムで時間が経過した後に完成品を受け取る。完成したアイテムは探索への持ち込みや霊魂への装備に使用できる。

製造だけでなく、手持ちの防具・盾・アクセサリーの**強化・ルーン（特殊効果）付与**も行える。鍛冶素材を消費してミニゲームに挑み、操作スコアに応じて強化の成功度やボーナスが変動する。

防具（鎧）は**全身一体型**として管理する（頭・胴・腕・脚などの部位分けなし）。3Dモデルの都合上、防具は1着で全身を覆うデザインとなるため、装備スロットは「防具：1枠」のみ。盾とアクセサリーはそれぞれ独立したスロットで管理する。

既製品（即時購入）は**道具屋**の役割。防具屋はあくまで「作る場所・育てる場所」として機能する。

## 基本情報

| 項目 | 内容 |
|---|---|
| **カテゴリ** | 探索準備・装備 |
| **建設タイプ** | 複数建設可能（最大3件） |
| **アンロック条件** | 鎧職人の霊魂をダンジョンで救出後にアンロック |

## 役割

### 製造できるアイテム種別

| カテゴリ | 説明 | 装備スロット |
|---|---|---|
| **防具（鎧）** | 全身一体型の鎧。軽鎧・重鎧・魔法衣など種類がある | 防具：1枠 |
| **盾** | 片手に装備する防御用装備。ブロック・耐性などの特殊効果を持つ | 盾：1枠 |
| **アクセサリー** | 指輪・お守り・ブローチなど。パッシブ効果・耐性・能力値ボーナスを付与 | アクセサリー：1枠 |

> 3種を同じ工房で製造できるため、防具屋1件あたりの生産物バリエーションが豊富になる。

### 製造フロー
1. **種別と設計図の選択**：防具・盾・アクセサリーのどれを作るかを選び、設計図を指定する
2. **素材の投入**：必要な素材（金属板・革・布・魔石・宝石など）を消費する
3. **生産開始**：製造キューに追加される。リアルタイムで時間が経過し、完成するまで待つ
4. **受け取り**：完成後にアイテムを受け取り、インベントリまたは保管庫へ格納

### 複数建設の効果
防具屋を複数建設すると**並列で製造できる数**が増加する。例えば3件建てれば3つのアイテムを同時に製造できる。

### 生産時間の目安

| レアリティ | 防具（鎧） | 盾 | アクセサリー |
|---|---|---|---|
| コモン | 5〜15分 | 5〜10分 | 3〜10分 |
| アンコモン | 30分〜1時間 | 20〜40分 | 15〜30分 |
| レア | 3〜6時間 | 2〜4時間 | 1〜3時間 |
| エピック | 12〜24時間 | 8〜16時間 | 6〜12時間 |
| クリア後（魔界） | 2〜3日 | 1〜2日 | 12〜24時間 |

> ※ 配置した霊魂のレベル・装備・施設レベル・時間短縮アイテムによって大幅に変動する。
> ※ アクセサリーは構造が単純なため、同レアリティでも防具より生産時間が短い。

---

### 防具・盾・アクセサリーの強化・ルーン付与

製造した装備（または既存の装備）を素材を消費してさらに育てる。強化の際はミニゲームが発生し、操作スコアに応じて成功度や付与ボーナスが変動する。

| フロー | 内容 |
|---|---|
| ① 対象選択 | 強化したい防具・盾・アクセサリーを選ぶ |
| ② 種別選択 | 「通常強化（防御力・HP上限などの数値アップ）」か「ルーン付与（耐性・特殊効果の付与）」かを選ぶ |
| ③ 素材消費 | 金属板・魔法の粉・魔石などを消費する |
| ④ ミニゲーム | 操作スコアに応じて強化の成功度・大成功ボーナスが変動する |

施設レベルが上がると扱える強化ランクとルーンの種類が増加し、より高い耐性・効果を持つ装備へと育てられる。

## 霊魂配置

> 霊魂の配置・変更はこの施設で直接行うか、**役場**の一括管理UIからも行うことができる。受け入れ可能なスロット数は施設レベルによって決まる。

| 項目 | 内容 |
|---|---|
| **配置スロット数** | Lv1：1体 → Lv3：2体 → Lv5(MAX)：3体 |
| **最適な武器種** | 槌（片手）または鎚（両手） |
| **武器の効果** | 生産速度アップ・防具のクオリティが上昇しやすくなる |
| **最適な防具** | 重鎧（プレートなど） |
| **防具の効果** | 同時製造キューの上限が増加する |
| **属性ボーナス** | 火属性のハンマー：製造した防具に炎耐性エンチャントが自動付与される確率アップ。鍛冶の火力が増し、製造時間が短縮される |
| **霊魂レベルの影響** | 霊魂のレベルが高いほど S・SS クオリティが出やすくなる（レベルと装備の両方が影響） |

> **差別化のポイント**：同じ設計図・素材を使っても、配置した霊魂のレベルと装備によって完成品のクオリティが大きく変わる。より良い防具を作りたければ、強い霊魂を救出して良い装備を持たせる必要がある。

## レベルアップ効果

| レベル | 解放内容・強化内容 |
|---|---|
| Lv1 | コモンの防具・盾・アクセサリーの製造が可能。基本強化（防御力の数値アップ）が可能。製造キュー数：1（並列1件まで） |
| Lv2 | アンコモンの製造が解放。全体の生産速度が上昇。コモンルーンの付与が解放 |
| Lv3 | レアの製造が解放。霊魂配置スロット+1。レアルーンの付与が解放 |
| Lv4 | エピックの製造が解放。製造キュー数増加（並列2件まで）。強化失敗時のペナルティ軽減。エピックルーンの付与が解放 |
| Lv5（MAX） | 全レアリティの製造が可能。品質上昇の確率が全体的にアップ。最高レアルーンの付与解放・複数ルーン同時付与可能。改造オプション解放 |

## クオリティシステム

完成したアイテムには C〜SS の5段階のクオリティランクが付く。アイテム種別ごとにクオリティの影響先が異なる。

### クオリティランク一覧

| ランク | 目安の出現率（通常工房・Lv1） | 防具（鎧）への影響 | 盾への影響 | アクセサリーへの影響 |
|---|---|---|---|---|
| C | 15% | 防御値が低い、スロットなし | ブロック率が低い | 効果値が最小 |
| B | 40% | 標準的な防御値 | 標準的なブロック率 | 標準的な効果値 |
| A | 30% | 高防御値。エンチャントスロット+1 | ブロック率アップ。特殊効果スロット+1 | 効果値が上位範囲 |
| S | 12% | 特殊耐性が稀に付与 | 特殊効果が稀に付与 | 追加パッシブ効果が稀に付与 |
| SS | 3% | 特殊耐性が確定付与 | 特殊効果が確定付与 | 追加パッシブ効果が確定付与 |

> **S・SS ランクの特殊付与例**
> - 防具（鎧）：炎ダメージ-30%、毒無効、被クリティカル率-20% など
> - 盾：攻撃をブロックした際に反撃ダメージ、ブロック時に霊素回復 など
> - アクセサリー：会心率+10%、移動コスト-1、ダンジョン入場時にHP微回復 など

### クオリティを決める4つの要素

| 要素 | 内容 |
|---|---|
| **① 霊魂のレベルと装備** | 配置霊魂のレベルが高く、高レアリティの武器を持たせるほど S・SS を引く確率が上昇（最重要） |
| **② 施設レベル** | レベルが上がるほど C ランクの出現率が下がり、S+SS 確率にボーナスが加算される |
| **③ 改造オプション** | 「品質特化工房」を選ぶと C・B が完全排除され、最低 A ランク保証になる |
| **④ 課金アイテム** | SS クオリティ確定（職人の魂結晶）や1ランク底上げ（品質の秘薬）が可能 |

---

## 改造オプション（Lv5 MAX後に解放）

> いずれか1つを選択。選択後は施設を解体しない限り変更不可。
> 複数建設している場合は施設ごとに別の改造オプションを選べる。

### 量産工房
生産速度が大幅アップし、製造キューの上限も増加する。クオリティの確率分布は C・B 寄りになるが、とにかく数をそろえたい運用スタイル向け。

### 品質特化工房
C・B ランクが出現しなくなり、最低 A ランク保証になる。同じ設計図・素材でも格段に良い防具が作れる。こだわりの一着を育てるスタイル向け。生産速度はやや低下。

### 耐性特化工房
耐性エンチャント（炎・氷・毒・麻痺・混乱など）が付与される確率が大幅アップする。特定のダンジョンバイオームやボス攻略に合わせた防具を作りやすくなる。通常の防御値は平均的。

### 強化特化工房
製造よりも強化・ルーン付与に特化した工房。強化時の防御系ステータスの強化値が×1.5倍になり、ルーン付与枠が+1される。ルーン付与の大成功率もアップ。製造機能は維持されるが生産速度はやや低下する。堅牢なビルドを深く追いたいプレイヤーに最適。

## 課金アイテムと無課金手段

### 通常手段（無課金）
| 手段 | 効果 |
|---|---|
| 霊魂のレベルアップ | S・SS の出現確率アップ（最重要） |
| 霊魂に武器装備 | レアリティに応じて S+SS 確率アップ |
| 施設レベルアップ | C ランク排除・S+SS 確率ボーナス加算 |
| 「品質特化工房」への改造 | C・B 完全排除、最低 A 保証 |
| 「耐性特化工房」への改造 | 特殊耐性付き防具の出現率大幅アップ |
| 料理屋のバフ料理 | 一時的な生産速度アップ（探索1回分） |

### 課金アイテム（有料）
| アイテム名（仮） | 効果 | 種別 |
|---|---|---|
| **職人の魂結晶** | 次の製造を SS クオリティ確定にする | クオリティ保証 |
| **品質の秘薬** | 次の製造のクオリティを1ランクアップ | クオリティ底上げ |
| **鍛冶の加速石** | 指定した製造中の防具を即時完成させる（クオリティに影響なし） | 時間短縮 |
| **工房の神炎** | 30分間、すべての製造速度を10倍にする | 時間短縮 |

> **設計上の注意**：課金要素は「クオリティ保証」と「時間短縮」に限定し、課金しないと入手できない防具・設計図は作らない。無課金でも全要素を揃えれば SS が25%前後になる設計を前提とする。

## 備考
- 防具（鎧）は**全身一体型（1枠）**のため、部位ごとの管理は行わない。3Dモデルとの整合性を保つ設計。
- 盾・アクセサリーも同じ防具工房で製造・強化できるため、施設1件あたりの活用幅が広い。
- **既製品（即時購入）は道具屋**が担当。防具屋は「こだわりの装備を作り・育てる」場所として差別化される。
- 設計図の入手方法：ダンジョンの宝箱・ボスドロップ・クエスト報酬・行商人。
- 強化ミニゲームの詳細仕様は `09_ミニゲームシステム/武器・防具屋の強化ゲーム.md` で管理。
- 強化に必要な素材（金属板・魔法の粉・魔石など）は `05_アイテム_インベントリシステム/資源・クラフト素材仕様.md` を参照。
