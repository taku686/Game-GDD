# 英雄廟（名誉の間）

## 概要
博物館の上位版として解放されるクリア後施設。歴代の挑戦記録・称号を展示するほか、討伐したボスの「魂のかけら」を奉納することでそのボスの特殊能力の一部を恒久解放できるアクティブなコレクション要素。

## 基本情報

| 項目 | 内容 |
|---|---|
| **カテゴリ** | クリア後解放 |
| **建設タイプ** | 1施設1つ固定（レベルアップ式） |
| **アンロック条件** | ストーリークリア後、最初のボスの「魂のかけら」を入手した後にアンロック |
| **重要度** | 中（長期やりこみの動機付け） |

## 役割
クリア後に特定のボスを倒すと稀に手に入る「魂のかけら」を英雄廟に奉納することで、そのボスの固有能力の一部が「恒久的なパッシブスキル」として解放される。

例：
- 「大竜（ファイアドレイク）の魂のかけら」を奉納 → 炎属性ダメージ+10%・炎耐性+5%が恒久バフとして付与される
- 「闇の王者（シャドウタイラント）の魂のかけら」を奉納 → 暗闇耐性+15%・クリティカル率+3%が恒久バフとして付与される

また、歴代の挑戦記録（最大到達階層・最速クリア時間・ウェーブ最高記録など）が展示され、プレイヤーの達成感を可視化する。

## 霊魂配置

> 霊魂の配置・変更はこの施設で直接行うか、**役場**の一括管理UIからも行うことができる。受け入れ可能なスロット数は施設レベルによって決まる。

| 項目 | 内容 |
|---|---|
| **配置スロット数** | Lv1：1体 → Lv3：2体 → Lv5(MAX)：3体 |
| **最適な武器種** | 剣（片手） |
| **武器の効果** | 英雄の意志が宿る剣を持たせることで、奉納した魂のかけらから解放される攻撃系パッシブスキルの効果値が上昇する。また、初めて奉納するボスには「初奉納ボーナス（追加パッシブ強化）」が付与される |
| **最適な防具** | 重鎧（プレートなど） |
| **防具の効果** | 歴戦の英雄の象徴たる重鎧を纏わせることで、奉納できるボスの種類が増加し、希少なボスの魂のかけらが入手しやすくなる確率が上がる |
| **属性ボーナス** | 光属性の剣：奉納した魂のかけらから得られる攻撃系パッシブスキルの種類が増加し、より多くの効果を同時に保持できるようになる |
| **レアリティの影響** | 高レアリティの剣を持たせるほど、特定のボスの魂のかけらが入手しやすくなる確率が上がる |

## レベルアップ効果

| レベル | 解放内容・強化内容 |
|---|---|
| Lv1 | 基本的な挑戦記録の展示機能が解放。魂のかけら奉納機能が使用可能に |
| Lv2 | 奉納できるボスの種類が増加。称号システムが解放 |
| Lv3 | 奉納したスキルの効果値が全体的に向上。霊魂配置スロット+1 |
| Lv4 | 複数の魂のかけらを合成して「伝説パッシブスキル」を生成できる機能が解放 |
| Lv5（MAX） | 全ボスの奉納が可能。伝説パッシブスキルの生成上限が増加。改造オプション解放 |

## 改造オプション（Lv5 MAX後に解放）

> いずれか1つを選択。選択後は施設を解体しない限り変更不可。

### 戦記廊（バトルハウス）
魂のかけら奉納で得られる「戦闘系パッシブスキル（ダメージ・耐性・クリティカル）」の効果値が強化特化になる。強くなることに一点集中したい戦闘特化プレイヤー向け。

### 記録の殿堂
挑戦記録の管理に特化し、達成した記録に応じて得られる「称号ボーナス」の種類と効果値が大幅に拡充される。NG+でも累積される記録が価値を持つため、長期的なやりこみを続けるほど恩恵が増し続ける。

## 備考
- クリア前の「博物館」は図鑑完成度によるパッシブバフが中心（積み重ねの成長感）。
- 英雄廟は「ボスを倒してその力を取り込む」というより能動的なコレクション要素として差別化される。
- 魂のかけらの入手率は確定ではなくドロップ品として設計（レアボスほど入手率が低い設計を想定）。
