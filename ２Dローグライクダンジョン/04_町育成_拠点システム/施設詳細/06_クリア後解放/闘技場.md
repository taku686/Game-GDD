# 闘技場（コロッセウム）

## 概要
魔王討伐後に解放されるクリア後施設。強力なモンスターのウェーブに挑むビルドテスト場。クリア後ダンジョン（魔界）への本格的な挑戦に向けて、ビルドを鍛え上げるための専用施設。

## 基本情報

| 項目 | 内容 |
|---|---|
| **カテゴリ** | クリア後解放 |
| **建設タイプ** | 1施設1つ固定（レベルアップ式） |
| **アンロック条件** | ストーリークリア（魔王討伐）後に建設可能 |
| **重要度** | 高（クリア後コンテンツの入口） |

## 役割
クリア後ダンジョン（魔界）攻略に向けた「ビルドテスト場」として機能する。通常のダンジョンとは異なる「ウェーブ制（次々と強敵が押し寄せる）」の戦闘システムで、プレイヤーのビルドが魔界に通用するかを試せる。

生き残ったウェーブ数に応じて報酬が増加するサバイバル形式のため、無限に稼ぎ続けることはできないが、毎回異なる敵の組み合わせで戦術の引き出しが試される。

「勇者クラス」解放後は、ここで全武器種のコンボを実際に試すことができる唯一の施設となる。

## 霊魂配置

> 霊魂の配置・変更はこの施設で直接行うか、**役場**の一括管理UIからも行うことができる。受け入れ可能なスロット数は施設レベルによって決まる。

| 項目 | 内容 |
|---|---|
| **配置スロット数** | Lv1：1体 → Lv3：2体 → Lv5(MAX)：3体 |
| **最適な武器種** | 任意（ビルドに合わせた武器種を選ぶ） |
| **武器の効果** | 配置した霊魂の武器種と同じ武器種を使うビルドで挑戦した場合、その武器の基本性能が+5%される「主将バフ」が得られる |
| **最適な防具** | 重鎧（プレートなど） |
| **防具の効果** | 競技場の管理・審判として機能。ウェーブ間の回復量がアップする |
| **属性ボーナス** | 施設に配置した霊魂の武器の属性に対応したモンスターが出現しにくくなる（弱点を突かれにくい） |
| **レアリティの影響** | 高レアリティの武器を持たせるほど、ウェーブ終了後の追加報酬（素材・ゴールド）の量が増加する |

## レベルアップ効果

| レベル | 解放内容・強化内容 |
|---|---|
| Lv1 | 基本的なウェーブ戦（10ウェーブ）が解放 |
| Lv2 | ウェーブ数の上限増加。魔界モンスターが出現するウェーブが追加 |
| Lv3 | 特定ボスが再登場する「ボスウェーブ」が解放。霊魂配置スロット+1 |
| Lv4 | ウェーブクリア報酬の質が向上 |
| Lv5（MAX） | 無制限ウェーブモードが解放。改造オプション解放 |

## 改造オプション（Lv5 MAX後に解放）

> いずれか1つを選択。選択後は施設を解体しない限り変更不可。

### 試練の闘技場
ウェーブ内の敵の強さが大幅アップし、より歯ごたえのある戦闘が楽しめるようになる。クリア時の報酬も大幅アップ。難易度が高い代わりに最高品質の素材・魔界限定装備が手に入りやすくなる。

### 訓練の闘技場
一度挑戦したウェーブを「リプレイ」できる機能が追加される。同じ構成でビルドを試行錯誤できるため、コンボの練習や特定の敵への対策を詰めたい場合に最適。報酬は低めだが繰り返し練習できる。
