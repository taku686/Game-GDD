# 転生の祭壇

## 概要
ニューゲームプラス（NG+）要素を提供するクリア後施設。一周クリア後に特定の条件を満たすと「転生」が可能になり、前周回の記録に基づいて拠点の一部レベルや特定の施設・成果を引き継いだ状態でゲームを再スタートできる。

## 基本情報

| 項目 | 内容 |
|---|---|
| **カテゴリ** | クリア後解放 |
| **建設タイプ** | 1施設1つ固定（レベルアップ式） |
| **アンロック条件** | ストーリークリア後、魔界の特定の深層に到達することで建設可能 |
| **重要度** | 中（長期的なやりこみの根幹） |

## 役割
「転生」を行うとゲームが最初からリスタートされるが、以下の要素が引き継がれる：

**引き継ぎ要素（案）：**
- 拠点の施設レベル（一部または全部、周回ごとに条件が増加）
- 図鑑・英雄廟の記録
- 習得した永続パッシブスキルの一部
- 特定の称号・実績

**引き継がれない要素：**
- キャラクターのレベル・装備
- ゴールド・素材
- ダンジョンの攻略記録（再度踏破が必要）

転生のたびに難易度が僅かに上がる一方で、引き継ぎ要素が増えていき「周回が重なるほど強くなれる」やりこみループが生まれる。

## 霊魂配置

> 霊魂の配置・変更はこの施設で直接行うか、**役場**の一括管理UIからも行うことができる。受け入れ可能なスロット数は施設レベルによって決まる。

| 項目 | 内容 |
|---|---|
| **配置スロット数** | Lv1：1体 → Lv3：2体 → Lv5(MAX)：3体 |
| **最適な武器種** | 盾（片手） |
| **武器の効果** | 転生の儀を守護する盾使いを配置することで、転生時に引き継げる施設レベルの数が増加する。強固な守りが前周回の積み重ねを次の周回へと確実に伝える |
| **最適な防具** | 軽鎧（レザーなど） |
| **防具の効果** | 身軽な装備が転生後の序盤の動きを助け、転生直後の「序盤加速バフ（ゴールドボーナス・素材集め効率↑）」の効果量が上昇する |
| **属性ボーナス** | 闇属性の盾：転生の深淵を守護する盾として、転生時に引き継げる永続パッシブスキルの数が1増加する |
| **レアリティの影響** | 高レアリティの盾を持たせるほど、転生時に引き継げる永続パッシブスキルの数がさらに増加する |

## レベルアップ効果

| レベル | 解放内容・強化内容 |
|---|---|
| Lv1 | 転生機能が解放。一部の施設レベルを引き継いで再スタート可能 |
| Lv2 | 引き継ぎ可能な施設・要素の種類が増加 |
| Lv3 | 転生後の「序盤加速バフ（ゴールドボーナス・素材集め効率アップ）」が強化。霊魂配置スロット+1 |
| Lv4 | 「選択的引き継ぎ」が解放（引き継ぎたい要素を自分で選べる） |
| Lv5（MAX） | 最大引き継ぎが可能に。転生ボーナスの質が最高に。改造オプション解放 |

## 改造オプション（Lv5 MAX後に解放）

> いずれか1つを選択。選択後は施設を解体しない限り変更不可。

### 強欲の転生
引き継ぎ要素を最大限に増やすことに特化。施設レベル・パッシブスキル・図鑑の記録がより多く引き継がれるようになる。その分、転生後の難易度上昇も大きくなるため、ハードコアなやりこみ向け。

### 無垢の転生
引き継ぎ要素を意図的に少なくし、最小限の恩恵でゲームをリスタートする「ほぼ初期状態」に近い転生。ただし「転生ボーナス（特殊な初期バフ）」が非常に強力になり、縛りプレイ的な楽しみ方ができる。
