# 国王の謁見所

## 概要
クリア後に城の内部が解放されることで機能する施設。NPCたちが人間の肉体を取り戻した後の「会話・絆システム」の舞台であり、特定のNPCと親密度を深めることで専用の強化やタッグ探索が解放される。

## 基本情報

| 項目 | 内容 |
|---|---|
| **カテゴリ** | クリア後解放 |
| **建設タイプ** | 1施設1つ固定（城の解放と同時に機能、別途建設は不要） |
| **アンロック条件** | ストーリークリア後、城の内部解放クエストを完了した後に機能開始 |
| **重要度** | 低（世界観的な深みとやりこみ要素の追加） |

## 役割
クリア後は霊魂たちが人間の肉体を取り戻しており、謁見所では彼らと直接会話しながら「絆システム」を進行させることができる。

**絆システムの内容：**
- 特定のNPCと会話・贈り物・共同クエストをこなすことで親密度が上昇
- 親密度が一定値を超えると「絆スキル」が解放される（そのNPCの職業に関連した特殊バフ）
- 最高親密度になると「タッグ探索」が解放される（そのNPCを相棒としてダンジョンに連れて行ける）

また、ここでは累積した拠点の発展状況・称号・達成記録を「国王に報告」する形式で、ゲームの進行具合を視覚的に振り返ることができる。

## 霊魂配置

> 霊魂の配置・変更はこの施設で直接行うか、**役場**の一括管理UIからも行うことができる。受け入れ可能なスロット数は施設レベルによって決まる。

| 項目 | 内容 |
|---|---|
| **配置スロット数** | Lv1：2体 → Lv3：4体 → Lv5(MAX)：6体 |
| **最適な武器種** | 剣（片手）・盾（片手）・扇/楽器（片手）などバランスよく配置 |
| **武器の効果** | 配置したNPCの職業に合わせた武器により、その職業に関連する絆スキルの効果値が上昇する |
| **最適な防具** | 平服（礼服・騎士服など、キャラクターに合ったもの） |
| **防具の効果** | 格式ある衣装で謁見所の雰囲気が向上し、絆進行速度が全体的にアップする |
| **属性ボーナス** | 光属性の武器：謁見所に来訪するNPCの種類が増え、より多くの絆を育てるチャンスが生まれる |
| **レアリティの影響** | 高レアリティの装備を持たせるほど、「タッグ探索」中の相棒NPCの能力値が底上げされる |

## レベルアップ効果

| レベル | 解放内容・強化内容 |
|---|---|
| Lv1 | 基本的な会話・絆システムが解放。3人のNPCと絆を育てることが可能 |
| Lv2 | 絆を育てられるNPCの数が増加。贈り物システムが解放 |
| Lv3 | 「タッグ探索」機能が解放。絆スキルの効果値がアップ。霊魂配置スロット+1 |
| Lv4 | タッグ探索時の相棒の能力が向上。特別な共同クエストが解放される |
| Lv5（MAX） | 全NPCと絆を育てることが可能に。最高親密度の絆スキルが解放。改造オプション解放 |

## 改造オプション（Lv5 MAX後に解放）

> いずれか1つを選択。選択後は施設を解体しない限り変更不可。

### 英雄の間
タッグ探索に特化した強化を行う。相棒NPCの戦闘能力（攻撃力・耐久性・特殊スキル）が大幅にアップし、二人での探索がより実戦的になる。絆システムの進行速度は標準的。

### 交流の庭
絆システムの進行速度とイベントの豊富さに特化した穏やかな空間に転換する。絆を深めるスピードが大幅アップし、隠しイベント・隠し会話の発生確率が上昇する。NPCたちの物語を掘り下げたいプレイヤー向け。

## 備考
- NPCシステムの詳細（霊魂救出からの絆進行フロー）は `07_キャラクター管理システム/NPCシステム.md` で管理予定（未着手）。
- 「人間に戻ったNPCたち」という設定を活かした固有の会話シナリオが、世界観への没入感を高める重要な要素となる。
- タッグ探索の戦闘仕様はゲームバランス上慎重に設計が必要（強すぎると主人公の存在感が薄れる）。
