# 霊魂の泉

## 概要
祈祷所の上位版として解放されるクリア後施設。クリア後限定の特殊素材を使うことで、通常の祈祷所では得られない強力なバフを付与できる。「勇者バフ」など探索の根本を覆すような極端な強化も選択可能。

## 基本情報

| 項目 | 内容 |
|---|---|
| **カテゴリ** | クリア後解放 |
| **建設タイプ** | 1施設1つ固定（レベルアップ式） |
| **アンロック条件** | ストーリークリア後、魔界の素材「霊力の雫」を初めて入手した後にアンロック |
| **重要度** | 中（クリア後の強化サイクルの中核） |

## 役割
祈祷所が「ゴールドやアイテムを奉納してランダムバフを得る基本施設」であるのに対し、霊魂の泉は「クリア後限定の特殊素材を使って強力なバフを得る上位施設」として機能する。

祈祷所との違い：
- 使用素材：通常ゴールド・アイテム → 霊力の雫・魔界素材
- バフ強度：標準的なバフ → 強力〜極端なバフ
- 選択肢：一部ランダム → より多くの固定選択肢から選べる

「勇者バフ」の例：
- 「今回の探索中、全武器種の会心率を+30%する」
- 「今回の探索中、霊素（スタミナ）の消費量が半分になる」

## 霊魂配置

> 霊魂の配置・変更はこの施設で直接行うか、**役場**の一括管理UIからも行うことができる。受け入れ可能なスロット数は施設レベルによって決まる。

| 項目 | 内容 |
|---|---|
| **配置スロット数** | Lv1：1体 → Lv3：2体 → Lv5(MAX)：3体 |
| **最適な武器種** | 槍/ランス（両手） |
| **武器の効果** | 泉を守護する槍使いを配置することで、霊力の雫の使用効率が上がり、同じ素材でも付与されるバフの効果値が全体的に上昇する |
| **最適な防具** | 魔法衣（ローブなど） |
| **防具の効果** | 霊素との親和性が高い依り代として、選択できるバフの選択肢数が+1される |
| **属性ボーナス** | 水属性の槍：泉の霊水と共鳴し、回復系・霊素節約系バフの効果値が大幅にアップする（×1.5） |
| **レアリティの影響** | 高レアリティの槍を持たせるほど、「勇者バフ」級のレアなバフの出現確率が上がる |

## レベルアップ効果

| レベル | 解放内容・強化内容 |
|---|---|
| Lv1 | 霊力の雫を使った強力バフ付与が解放（祈祷所より強力） |
| Lv2 | バフの種類が増加。魔界専用バフが解放される |
| Lv3 | 勇者バフ級の極端な強化バフが解放。霊魂配置スロット+1 |
| Lv4 | 2つ同時バフ付与が可能に。バフ効果値が全体的に上昇 |
| Lv5（MAX） | 全バフ種解放。「泉の祝福（3バフ同時付与）」が使用可能に。改造オプション解放 |

## 改造オプション（Lv5 MAX後に解放）

> いずれか1つを選択。選択後は施設を解体しない限り変更不可。

### 覇道の泉
攻撃・スピード・クリティカル系の「攻める」バフに特化。これらのバフの効果値が×2倍になる。「今回の探索は全力で攻め切る」というプレイスタイルを全力で後押しする。防御・回復系バフは選択肢から外れる。

### 守護の泉
耐性・防御・霊素節約系の「耐える」バフに特化。深層の長時間探索で死なないために必要なバフが揃う。攻撃系バフは選択肢から外れるが、「とにかく生き延びる」プレイスタイルを最大限にサポートする。

## 備考
- 祈祷所は引き続き使用可能。「祈祷所で基本バフ + 霊魂の泉で強力バフ」のように両施設を組み合わせる運用が想定される。
- 霊力の雫はクリア後ダンジョン（魔界）専用のドロップ素材。クリア前には入手不可。
