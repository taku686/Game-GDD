# 魔界研究所

## 概要
クリア後ダンジョン（魔界）専用の強化施設。魔界で集めた「瘴気の結晶」を研究することで、魔界探索に必要な専用バフ・耐性を解放し、より深い階層へのアクセスを可能にする。

## 基本情報

| 項目 | 内容 |
|---|---|
| **カテゴリ** | クリア後解放 |
| **建設タイプ** | 1施設1つ固定（レベルアップ式） |
| **アンロック条件** | ストーリークリア後、最初に魔界の1階層を踏破した後にアンロック |
| **重要度** | 高（クリア後コンテンツの深化に直結） |

## 役割
魔界ダンジョンで入手できる特殊素材「瘴気の結晶」を消費して、魔界専用の永続バフ・耐性を解放する。

この施設がなければ魔界の深層（高瘴気地帯）に長時間留まることができないため、クリア後コンテンツを本格的に進めるには必須の施設となる。

魔界のバフ・耐性の例：
- 瘴気耐性（魔界独自のダメージ軽減）
- 魔界モンスターへのダメージボーナス
- 魔界ドロップ品のレアリティ底上げ
- 深層転移の解放（特定の研究進捗で転移先が解放される）

## 霊魂配置

> 霊魂の配置・変更はこの施設で直接行うか、**役場**の一括管理UIからも行うことができる。受け入れ可能なスロット数は施設レベルによって決まる。

| 項目 | 内容 |
|---|---|
| **配置スロット数** | Lv1：1体 → Lv3：2体 → Lv5(MAX)：3体 |
| **最適な武器種** | 鎌（両手） |
| **武器の効果** | 瘴気を刈り取る大鎌を持たせることで、魔界での瘴気の結晶の採取効率が上昇し、研究1回あたりの消費コストが削減される |
| **最適な防具** | 魔法衣（ローブなど） |
| **防具の効果** | 魔界の知識に精通した依り代として、研究した魔界情報の精度が上がり、より効果的なバフが得られる |
| **属性ボーナス** | 闇属性の鎌：魔界探索中に瘴気の結晶を追加で1個確定回収できる。魔界モンスターへのダメージボーナス研究の効果値も上昇する |
| **レアリティの影響** | 高レアリティの鎌を持たせるほど、魔界専用装備（特殊エンチャント付き）のレシピ解放速度が上がる |

## レベルアップ効果

| レベル | 解放内容・強化内容 |
|---|---|
| Lv1 | 基本的な瘴気耐性の研究が可能。浅い魔界の探索が安定する |
| Lv2 | 魔界モンスター対策バフの研究が解放。中間層の攻略が可能に |
| Lv3 | 魔界ドロップ品のレアリティ向上研究が解放。霊魂配置スロット+1 |
| Lv4 | 深層転移先の解放研究が可能になる。高難度エリアへのアクセスが開ける |
| Lv5（MAX） | 全研究ツリーが解放。魔界の最深部への挑戦が可能に。改造オプション解放 |

## 改造オプション（Lv5 MAX後に解放）

> いずれか1つを選択。選択後は施設を解体しない限り変更不可。

### 攻性魔界研究
魔界モンスターへのダメージボーナスと攻撃系バフの効果値に研究リソースを集中させる。魔界の敵を効率よく倒し、より深く・より素早く探索を進めたい攻撃型プレイヤーに向く。防御・耐性系の研究効率はやや低下する。

### 防性魔界研究
瘴気耐性・魔界モンスターからのダメージ軽減・状態異常耐性の効果値に研究リソースを集中させる。安全に長時間深層で活動できる堅実な探索スタイルを実現する。攻撃系バフの研究効率はやや低下する。

## 備考
- クリア後ダンジョンの設定（「肉体を取り戻した後も魂を鎧に飛ばして探索する」）は `01_ストーリー_進行システム/世界観とストーリーフロー.md` を参照。
- 瘴気の結晶は魔界専用の素材で、クリア前のダンジョンでは入手不可。
