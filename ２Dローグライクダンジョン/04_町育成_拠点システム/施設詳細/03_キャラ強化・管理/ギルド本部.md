# ギルド本部

## 概要
職業の転職操作・上位職/マスター職のアンロック・永続パッシブスキルの解放・スキルポイントの振り直しを担う施設。キャラクターの育成・職業管理に完全特化した施設。クエストの受注は酒場が担当する。

## 基本情報

| 項目 | 内容 |
|---|---|
| **カテゴリ** | キャラ強化・管理 |
| **建設タイプ** | 1施設1つ固定（レベルアップ式） |
| **アンロック条件** | ギルドマスターの霊魂をダンジョンで救出後にアンロック |

## 役割
主に4つの機能を担う。

1. **職業変更（転職）**：ギルドマスターに話しかけることで、すでに解放済みの職業（クラス）に自由に転職できる（ドラゴンクエスト方式）。ゴールドを消費する。転職時にはスキルポイントがリセットされ、新職業のスキルツリーへ振り直しが必要になる。
2. **上位職・マスター職のアンロック**：施設レベルと特定の条件を満たすことで新たな「上位職」や「マスター職」をアンロックする。アンロックした職業への転職操作も本施設で行う。
3. **永続パッシブスキルの解放**：ゲームオーバー時にも引き継がれる恒久的なパッシブスキルを習得・強化する（メタプログレッションの核）。
4. **スキルポイントの振り直し**：現在の職業のスキルポイント割り振りをリセットして再配分できる。ゴールドまたは特定素材を消費する（施設Lv5以降は無料）。

> 酒場 = 住人依頼クエスト受注 ＋ NPC交流 ＋ ギャンブル
> ギルド本部 = 職業変更（転職）＋ 上位職・マスター職のアンロック ＋ 永続スキル解放 ＋ スキル振り直し

## 霊魂配置

> 霊魂の配置・変更はこの施設で直接行うか、**役場**の一括管理UIからも行うことができる。受け入れ可能なスロット数は施設レベルによって決まる。

| 項目 | 内容 |
|---|---|
| **配置スロット数** | Lv1：1体 → Lv3：2体 → Lv5(MAX)：3体 |
| **最適な武器種** | 剣（片手） |
| **武器の効果** | ギルドマスターの権威を示す剣を持たせることで、上位職・マスター職のアンロック条件が全体的に緩和される。さらに転職コストが低下する |
| **最適な防具** | 軽鎧（レザーなど） |
| **防具の効果** | 現役の冒険者スタイルがギルドの信頼感を高め、スキルポイントの振り直しコストが低下する。永続パッシブスキルの解放コストも若干下がる |
| **属性ボーナス** | 光属性の剣：新たな上位職のアンロック条件が一部緩和される |
| **レアリティの影響** | 高レアリティの剣を持たせるほど、永続スキルポイントの獲得効率が上がる |

## レベルアップ効果

| レベル | 解放内容・強化内容 |
|---|---|
| Lv1 | 基本職5種への転職が可能。スキルポイント振り直し機能が解放。初期パッシブスキルリストが解放 |
| Lv2 | 転職コストが低下。上位職アンロックの条件が一部緩和 |
| Lv3 | 上位職（10種）へのアンロック条件が解放開始。霊魂配置スロット+1 |
| Lv4 | 高レア職業（マスター職前段）のアンロック解放。永続スキルの解放コストが低下 |
| Lv5（MAX） | マスター職・勇者職へのアンロック条件が解放。転職・スキル振り直しが無料に。改造オプション解放 |

## 改造オプション（Lv5 MAX後に解放）

> いずれか1つを選択。選択後は施設を解体しない限り変更不可。

### 上位職特化ギルド
上位職・マスター職への転職条件が全体的に緩和される。通常では難しい条件（「〇〇スキルLv10以上」など）のハードルが下がり、幅広いビルド構成を素早く試せるようになる。転職コストはやや上昇する。

### スキルリセット館
永続パッシブスキルの振り直しが低コストで行えるようになる。通常は高価なコストが必要なリセットが大幅に割安になり、試行錯誤のビルド構築を後押しする。スキルポイントの振り直しコストも全体的に低下する。

### 修行の道場
転職後の習熟度上昇速度が大幅にアップする。新たな職業でのスキル習得・レベルアップに必要な経験値が低下し、マルチクラスビルドの構築を後押しする。転職コストは標準的なまま。

## 備考
- 職業システムの詳細は `06_職業_ビルド構築システム/職業システム概要案.md` を参照。
- 転職コストはゴールドを消費する（施設Lv5以降は無料）。転職すると現在のスキルポイントはリセットされ、新職業のスキルツリーへの振り直しが必要。
- スキルポイントの振り直しはゴールドまたは特定素材を消費する（施設Lv5以降は無料）。
- クエストの受注は酒場で行う。住人からの依頼や特殊イベントクエストは酒場が担当。
