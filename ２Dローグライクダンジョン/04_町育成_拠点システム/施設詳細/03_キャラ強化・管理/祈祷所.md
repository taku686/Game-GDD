# 祈祷所（神殿・祠）

## 概要
探索に出発する前に資金やアイテムを奉納し、ランダムなバフを次回の探索1回分だけ付与される施設。クリア後に上位版「霊魂の泉」が解放される前提で設計された基本バフ施設。

## 基本情報

| 項目 | 内容 |
|---|---|
| **カテゴリ** | キャラ強化・管理 |
| **建設タイプ** | 1施設1つ固定（レベルアップ式） |
| **アンロック条件** | 役場をLv1以上にアップグレード後に建設可能 |

## 役割
出発前にゴールドや特定のアイテムを「奉納」することで、その探索1回限りのランダムバフを付与してもらう。バフの内容は「ドロップ率アップ」「獲得経験値増加」「罠耐性アップ」「属性ダメージ軽減」など多種多様。

奉納する金額・アイテムの価値が高いほど、より強力なバフを引く確率が上がる仕組み。

## 霊魂配置

> 霊魂の配置・変更はこの施設で直接行うか、**役場**の一括管理UIからも行うことができる。受け入れ可能なスロット数は施設レベルによって決まる。

| 項目 | 内容 |
|---|---|
| **配置スロット数** | Lv1：1体 → Lv3：2体 → Lv5(MAX)：3体 |
| **最適な武器種** | 扇/楽器（片手） |
| **武器の効果** | 神への奉納を演出し、バフの発動確率とバフ効果値が上昇する |
| **最適な防具** | 魔法衣（ローブなど） |
| **防具の効果** | 霊素との親和性が高い依り代として機能し、強力なバフを引く確率が上がる |
| **属性ボーナス** | 光属性の扇：バフの種類が1つ追加で提示され、選択肢が増える（通常はランダム1択→3択から選べるようになる） |
| **レアリティの影響** | 高レアリティの扇/楽器を持たせるほど、最上位バフ（通常では滅多に引けないもの）の出現確率が上がる |

## レベルアップ効果

| レベル | 解放内容・強化内容 |
|---|---|
| Lv1 | 基本バフ（3〜5種）がランダムで付与される |
| Lv2 | バフの種類が増加。奉納コストが低下 |
| Lv3 | バフ効果値が上昇。霊魂配置スロット+1 |
| Lv4 | 上位バフ種が解放される。奉納なしでも弱いバフが1つ得られる |
| Lv5（MAX） | 全バフ種が解放。奉納量に応じてバフ数が増加（最大2つ同時）。改造オプション解放 |

## 改造オプション（Lv5 MAX後に解放）

> いずれか1つを選択。選択後は施設を解体しない限り変更不可。

### 霊力増幅祠
バフの効果値が全体的に×1.5倍に増幅される。高い奉納コストと引き換えに、他では得られない強力なバフ効果を実現する。ハイリスク・ハイリターン路線。奉納コストも×1.5倍増加する。

### 複数祝福祠
受けられるバフ数が増加する（通常2つ→最大3〜4つ）。各バフの効果値は標準的だが、複数のバフを重ねることで探索全体を底上げできる。幅広いサポートを好むプレイヤーに向く。

### 選択祝福祠
ランダム付与からの脱却。奉納後に提示される選択肢が増え（3択→5択）、その中から自分で選べる確率が上がる。完全な固定化ではないが、偶然に左右されずに計画的なビルド補強ができる。

## 備考
- クリア後に上位版「霊魂の泉」（06_クリア後解放）が解放される。祈祷所はその前段階として設計されており、クリア後は「より強力なバフが得られる霊魂の泉」に役割の中心が移る。
- 奉納するアイテムの設定によっては「使わなかった消耗品」や「余った素材」を奉納することも可能にすると、無駄なく活用できるループが生まれる。
