# 役場（建築所）

## 概要
町全体の管理と、新たな施設の建設・アップグレードを統括する中心施設。ダンジョンから持ち帰った建築素材と資金を消費して他のすべての施設をレベルアップさせる、メタプログレッションの要。

## 基本情報

| 項目 | 内容 |
|---|---|
| **カテゴリ** | 町発展・システム |
| **建設タイプ** | 1施設1つ固定（初期配置施設。解体不可） |
| **アンロック条件** | ゲーム開始時から配置済み |

## 役割
主に2つの機能を担う。

1. **施設の建設・レベルアップ管理**：すべての施設の建設・アップグレードを「指示」する管理拠点。役場のレベルが上がるほど建設できる施設の種類が増え、各施設の最高レベル上限も上昇する（役場Lvが施設Lvの上限を規定する仕組みも検討）。建設中・アップグレード中の施設はリアルタイムで時間が経過し、完了まで一定の待ち時間が発生する（配置と開拓システム参照）。

2. **霊魂配置の一括管理UI**：町内のすべての施設への霊魂の配置・変更・取り外しを役場のUIから一括で確認・操作できる。霊魂の配置は各施設でも直接行えるが、役場では全施設の配置状況を俯瞰しながら効率よく調整できる。各施設で受け入れ可能な霊魂の数（スロット数）は施設のレベルによって決まる。施設ごとに推奨される武器種・防具・属性が異なるため、目的に合わせた配置が戦略の鍵となる。

## 霊魂配置（役場自体への配置）

| 項目 | 内容 |
|---|---|
| **配置スロット数** | Lv1：1体 → Lv3：2体 → Lv5(MAX)：3体 |
| **最適な武器種** | ハンマー（両手） |
| **武器の効果** | 施設建設・アップグレードの時間が短縮される（建築コスト削減にも貢献） |
| **最適な防具** | 重鎧（プレートなど） |
| **防具の効果** | 重作業の耐久性アップ。複数施設の同時建設・監督が可能になる（並列建設数が増加） |
| **属性ボーナス** | 地属性のハンマー：建設に使う建築素材の消費量が削減される |
| **レアリティの影響** | 高レアリティのハンマーを持たせるほど、建設完了時に「施設強化ボーナス（スペック底上げ）」が付く確率がアップ |

## レベルアップ効果

| レベル | 解放内容・強化内容 |
|---|---|
| Lv1 | 基本施設（武器屋・防具屋・道具屋・役場など）の建設が可能 |
| Lv2 | 解放される施設の種類が増加。建設速度が上昇 |
| Lv3 | 施設レベルの上限が解放。並列建設が可能になる。霊魂配置スロット+1 |
| Lv4 | 高度な施設（ギルド本部・調合所など）の建設が解放。建築コストが削減 |
| Lv5（MAX） | 全施設が建設可能に。最大並列建設数が増加。改造オプション解放 |

## 改造オプション（Lv5 MAX後に解放）

> いずれか1つを選択。選択後は施設を解体しない限り変更不可。

### 建設省（スピード特化）
施設の建設・アップグレードにかかるリアルタイム待ち時間が全体的に短縮される。建設ラッシュで町を一気に発展させたい時に最大の力を発揮する。素材コストは変わらない。

### 資材省（コスト特化）
施設の建設・アップグレードに必要な建築素材とゴールドコストが全体的に削減される。素材が不足しがちな序盤〜中盤に特に有効で、少ない資源で多くの施設を育てることができる。建設時間は変わらない。

### 開拓省（土地特化）
未開の地の解放（開拓）にかかるコストと時間が削減され、開拓ボーナス（レア素材・設計図など）が見つかりやすくなる。町の土地を広げながら発展を進めたいプレイヤーに向く。

## 備考

- 施設の建設・アップグレードシステムの詳細は `04_町育成_拠点システム/配置と開拓システム.md` を参照。
