# 探検隊本部（傭兵ギルド）

## 概要
使っていない別クラスのキャラクターや牧場のペットたちを「自動素材収集」に派遣するシステムの拠点。プレイヤーがメインキャラでダンジョンを探索している間に、自動的にアイテムや素材を収集してくれる放置・自動化要素。

## 基本情報

| 項目 | 内容 |
|---|---|
| **カテゴリ** | 町発展・システム |
| **建設タイプ** | 1施設1つ固定（レベルアップ式） |
| **アンロック条件** | 2人目の霊魂を救出後に建設可能 |

## 役割
プレイヤーが操作しないサブキャラクター・ペットを探索任務に派遣し、戻ってきた時に素材・ゴールドを持ち帰る「自動収集システム」を提供する。

派遣先は「ダンジョンの特定の階層」「特定のバイオーム」などを指定でき、目的の素材に合わせた任務を組める。派遣中のキャラクターはメインの探索には使えない。

## 霊魂配置

> 霊魂の配置・変更はこの施設で直接行うか、**役場**の一括管理UIからも行うことができる。受け入れ可能なスロット数は施設レベルによって決まる。

| 項目 | 内容 |
|---|---|
| **配置スロット数** | Lv1：1体 → Lv3：2体 → Lv5(MAX)：3体 |
| **最適な武器種** | 弓（両手） |
| **武器の効果** | 派遣したキャラクターの収集効率アップ（より多くの素材を持ち帰る） |
| **最適な防具** | 軽鎧（レザーなど） |
| **防具の効果** | 素早い行動で派遣から帰還するまでの時間が短縮される |
| **属性ボーナス** | 風属性の弓：派遣先でレアな素材を発見する確率が大幅にアップ |
| **レアリティの影響** | 高レアリティの弓を持たせるほど、派遣したキャラクターが負傷・失敗するリスクが低下する |

## レベルアップ効果

| レベル | 解放内容・強化内容 |
|---|---|
| Lv1 | 1体のキャラクターを派遣可能。基本的な素材収集任務のみ |
| Lv2 | 同時派遣数が2体に増加。任務の種類が増加 |
| Lv3 | 同時派遣数が3体に増加。レア素材専門任務が解放。霊魂配置スロット+1 |
| Lv4 | 同時派遣数が4体に増加。深層の任務が解放（高リスク・高リターン） |
| Lv5（MAX） | 同時派遣数が5体に増加。改造オプション解放 |

## 改造オプション（Lv5 MAX後に解放）

> いずれか1つを選択。選択後は施設を解体しない限り変更不可。

### 精鋭遠征隊
派遣するキャラクターの能力値（装備・レベル）が高いほど収集してくる素材のレアリティが高くなる比例計算が強化される。強いキャラを派遣するほどリターンが大きくなる「実力主義」型の運用に最適。

### 大規模遠征
同時派遣できるキャラクターの数がさらに増加（+2体）する。個々の収集効率は標準的だが、数の力で総収集量を最大化できる。多くのサブキャラクターを育てていると特に有効。

### 情報収集隊
素材収集よりも「ダンジョン情報（特定の敵の出現率・ボスの弱点・隠し部屋の場所など）」の収集に特化する。事前情報を持った状態でメインキャラが探索に臨めるため、難しいダンジョンの攻略を間接的に補助する。

## 備考
- 派遣中のキャラクターには装備を持たせることができ、装備の質が収集結果に影響する。
- 派遣任務のリスク設定（失敗時のペナルティなど）は未決定。高リスク任務では装備を消耗する可能性も検討中。
