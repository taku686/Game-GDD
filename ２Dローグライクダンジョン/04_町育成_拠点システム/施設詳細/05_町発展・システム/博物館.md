# 博物館（図書館・記録所）

## 概要
遭遇したモンスター・拾ったアイテム・釣った魚・訪れた地などの「図鑑」を展示・記録していく施設。図鑑の完成度が上がるごとに町全体や全キャラクターへの恒久的なパッシブバフが得られるコレクション要素の中核。クリア後に上位版「英雄廟」が追加される。

## 基本情報

| 項目 | 内容 |
|---|---|
| **カテゴリ** | 町発展・システム |
| **建設タイプ** | 1施設1つ固定（レベルアップ式） |
| **アンロック条件** | 役場をLv2以上にアップグレード後に建設可能 |

## 役割
ダンジョン探索・スローライフ・戦闘などすべての行動が「記録」として残り、完成度が一定値を超えると恒久的なパッシブバフが解放される。

具体的な図鑑の種類：
- **モンスター図鑑**：倒したモンスターの種類・討伐数・生態情報
- **アイテム図鑑**：発見・使用したアイテムの情報（収集種数をカウント）
- **魚類図鑑**：釣り堀・ダンジョンで得た魚の情報（収集種数をカウント）
- **地形・バイオーム図鑑**：訪れたバイオームや特殊地形の記録

## 霊魂配置

> 霊魂の配置・変更はこの施設で直接行うか、**役場**の一括管理UIからも行うことができる。受け入れ可能なスロット数は施設レベルによって決まる。

| 項目 | 内容 |
|---|---|
| **配置スロット数** | Lv1：1体 → Lv3：2体 → Lv5(MAX)：3体 |
| **最適な武器種** | 弓（両手） |
| **武器の効果** | 遠方からモンスターや生態系を安全に観察・採集できる弓を持たせることで、図鑑エントリーの新規登録時にパッシブバフの解放閾値が低下する（少ない収集数でバフを解放しやすくなる）。特定のモンスターの弱点情報が開示される確率も上昇する |
| **最適な防具** | 魔法衣（ローブなど） |
| **防具の効果** | 学術的な外見で来館者（プレイヤー）への情報提供が充実し、モンスターの弱点情報などが開示されやすくなる |
| **属性ボーナス** | 風属性の弓：遠方のモンスター生態を素早く観察でき、ダンジョン探索前に「モンスター弱点ヒント」が1種類無料で開示される。新規図鑑エントリー増加に応じてパッシブバフの解放閾値がより低下しやすくなる |
| **レアリティの影響** | 高レアリティの弓を持たせるほど、特定の図鑑を完成させた際の「解放報酬（特別なバフやアイテム）」の質が上がる |

## レベルアップ効果

| レベル | 解放内容・強化内容 |
|---|---|
| Lv1 | モンスター図鑑・アイテム図鑑が開設 |
| Lv2 | 魚類図鑑が開設。各図鑑バフの解放閾値が下がる（より早くバフが得られる） |
| Lv3 | 地形図鑑が開設。霊魂配置スロット+1 |
| Lv4 | 各図鑑のカテゴリが細分化され、よりきめ細かいバフが得られる |
| Lv5（MAX） | すべての図鑑が最大展示可能に。全バフの効果値+10% |

## モンスター図鑑：討伐数によるレアドロップ率上昇

モンスターごとに討伐数を記録し、数が増えるほど**そのモンスター専用のレアドロップ率**が段階的に上昇する。  
効果は**全プレイヤー・全ラン共通**のパッシブとして永続適用される。

### 討伐数ランク（モンスター種ごとに独立）

| ランク | 討伐数 | レアドロップ率ボーナス |
|---|---|---|
| 知識Ⅰ | 10体 | +3% |
| 知識Ⅱ | 30体 | +6% |
| 知識Ⅲ | 100体 | +10% |
| 知識Ⅳ | 300体 | +15% |
| 知識Ⅴ（最大） | 1000体 | +20% |

> **設計メモ**：レアドロップ率ボーナスはそのモンスターのドロップテーブルに対してのみ適用。上限は+20%で固定。ボスモンスターは討伐数の閾値を半分（例：5/15/50/150/500）に緩和する。

## アイテム図鑑：収集種数によるバフ

発見・入手したアイテムの**ユニーク種数**に応じて、全体パッシブバフが解放される。

| 収集種数 | バフ効果 |
|---|---|
| 10種 | ショップのアイテム価格 -5% |
| 25種 | ダンジョン内のアイテムドロップ量 +5% |
| 50種 | 装備のレアリティ底上げ（コモン→アンコモン出現率 +10%） |
| 100種 | クラフトコスト -10% |
| 200種 | 全装備の識別が不要になる（未鑑定品がなくなる） |
| 全種コンプリート | レジェンダリー装備のドロップ率 +5%（全ダンジョン共通） |

---

## 魚類図鑑：収集種数によるバフ

釣り堀・ダンジョン内で入手した魚の**ユニーク種数**に応じて、全体パッシブバフが解放される。

| 収集種数 | バフ効果 |
|---|---|
| 5種 | 釣りの成功率 +5% |
| 15種 | 魚を素材にした料理の効果時間 +20% |
| 30種 | 料理・食事バフの最大スタック数 +1 |
| 50種 | ダンジョン内の水源エリアで魚が自動収集される |
| 全種コンプリート | 釣り時に低確率でレア魚・伝説魚が追加ドロップ |
