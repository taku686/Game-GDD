# 酒場

## 概要
霊魂たち・住人NPCが集まる情報交換と依頼の場。町に暮らす様々な住人からのクエスト受注・NPC交流・ギャンブルミニゲームなどを提供する。職業の転職や上位職のアンロックはギルド本部が担当し、酒場は「気軽に立ち寄れる拠点の憩いの場」として機能する。

## 基本情報

| 項目 | 内容 |
|---|---|
| **カテゴリ** | 町発展・システム |
| **建設タイプ** | 1施設1つ固定（レベルアップ式） |
| **アンロック条件** | 最初の霊魂を救出後に建設可能 |

## 役割
主に3つの機能を担う。

1. **住人依頼クエスト受注**：町の住人NPCからの依頼クエストを受注できる。「〇〇を◯匹倒す」「〇〇のアイテムを納品」「特定の場所を探索する」など多様な依頼が集まる。施設レベルが上がるほど、より多様な住人が依頼を持ち込むようになる。報酬はゴールド・素材・施設の「設計図」など。
2. **NPC交流（絆システム前段）**：常駐しているNPCと会話でき、好感度を上げることができる。
3. **ギャンブルミニゲーム**：ゴールドを賭けて大きなリターンを狙える娯楽要素（ハイリスク・ハイリターン）。

> 酒場 = 住人依頼クエスト受注 ＋ NPC交流 ＋ ギャンブル
> ギルド本部 = 職業変更（転職）＋ 上位職・マスター職のアンロック ＋ 永続スキル解放 ＋ スキル振り直し

## 霊魂配置

> 霊魂の配置・変更はこの施設で直接行うか、**役場**の一括管理UIからも行うことができる。受け入れ可能なスロット数は施設レベルによって決まる。

| 項目 | 内容 |
|---|---|
| **配置スロット数** | Lv1：1体 → Lv3：2体 → Lv5(MAX)：3体 |
| **最適な武器種** | 爪/ナックル（片手） |
| **武器の効果** | 酒場の用心棒として爪/ナックルを持たせることで、ギャンブルミニゲームの不正防止が強化され、大勝ち時の報酬上限が上昇する。また悪徳な依頼人を弾くことでクエスト完了ボーナス報酬が付く確率がアップ |
| **最適な防具** | 平服（給仕服） |
| **防具の効果** | 人間らしい給仕姿で顧客満足度が上がる。クエスト完了時のボーナス報酬が付く確率アップ |
| **属性ボーナス** | 雷属性の爪/ナックル：酒場全体に気迫が満ち渡り、クエスト報酬が+20%される |
| **レアリティの影響** | 高レアリティの爪/ナックルを持たせるほど、特別イベント（レアNPC訪問・隠しクエスト解放）の発生確率が上がる |

## レベルアップ効果

| レベル | 解放内容・強化内容 |
|---|---|
| Lv1 | 基本的な住人依頼クエストが受注可能。NPC交流機能が解放 |
| Lv2 | クエストの種類が増加（依頼人の幅が広がる）。報酬が充実 |
| Lv3 | ギャンブルミニゲームが解放。霊魂配置スロット+1 |
| Lv4 | 中〜高難度の住人依頼が受注可能に。NPC絆イベントが発生しやすくなる |
| Lv5（MAX） | 全クエスト種類が解放。設計図系の特別報酬クエストが追加。改造オプション解放 |

## 改造オプション（Lv5 MAX後に解放）

> いずれか1つを選択。選択後は施設を解体しない限り変更不可。

### 依頼人ギルド
クエストの質・量・報酬レベルが全体的に向上する。レアリティの高いクエスト（設計図・特殊素材が報酬のもの）の出現率がアップし、クエスト完了報酬が×1.5倍になる。

### 賭博場
ギャンブルミニゲームの種類が大幅に増加し、より高リスク・高リターンの賭けが可能になる。大勝ちの際の報酬上限が大幅に跳ね上がる。通常クエストの報酬は標準的なまま。

### 霊魂の溜まり場
NPC絆システムの進行速度が大幅アップする。特別な絆イベントが発生しやすくなり、絆を深めたNPCから特別な能力解放や専用クエストが得られる。クエスト報酬は標準的。
