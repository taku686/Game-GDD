# 行商人誘致所

## 概要
帰還するたびに出現確率が変動する「行商人（訪問NPC）」を呼び込むための施設。建設することで行商人の出現率・品揃えレベルが向上する。行商人自体は常設でなくランダムイベントとして登場する。

## 基本情報

| 項目 | 内容 |
|---|---|
| **カテゴリ** | 町発展・システム |
| **建設タイプ** | 1施設1つ固定（レベルアップ式） |
| **アンロック条件** | 役場をLv2以上にアップグレード後に建設可能 |

## 役割
「行商人」は帰還のたびにランダムで出現し、通常の武器屋・道具屋には並ばない激レア装備・特別な種・変わった設計図などを販売する特殊ショップ。

行商人誘致所を建設・アップグレードすることで：
- 行商人の出現確率がアップする
- 出現する行商人の品揃えレベルが向上する
- 特殊な専門行商人（魔道具屋・錬金術師など）が出現しやすくなる

以前の「闇市（常設施設）」の設計から、「行商人というイベントを誘致する施設」に変更することで、常設施設として存在することへの違和感を解消している。

## 霊魂配置

> 霊魂の配置・変更はこの施設で直接行うか、**役場**の一括管理UIからも行うことができる。受け入れ可能なスロット数は施設レベルによって決まる。

| 項目 | 内容 |
|---|---|
| **配置スロット数** | Lv1：1体 → Lv3：2体 |
| **最適な武器種** | 剣（片手）または盾（片手） |
| **武器の効果** | 護衛役として機能。行商人との取引中のアイテムロスト率が低下する |
| **最適な防具** | 平服（商人風の服装） |
| **防具の効果** | 商人らしい外見で行商人との交渉が有利になり、一部商品で値引きが発生しやすくなる |
| **属性ボーナス** | 風属性の剣/盾：商路の風を呼び込み、帰還のたびに特殊行商人（レア品専門）が出現する確率が大幅アップ。旅の出会いが増える |
| **レアリティの影響** | 高レアリティの武器を持たせるほど、行商人が持ってくるアイテムの最大レアリティが上昇する |

## レベルアップ効果

| レベル | 解放内容・強化内容 |
|---|---|
| Lv1 | 帰還時に行商人が出現する確率が解放される（低確率） |
| Lv2 | 行商人の出現確率がアップ。品揃えの質が向上 |
| Lv3 | 専門行商人（武具専門・消耗品専門など）が出現するようになる。霊魂配置スロット+1 |
| Lv4 | エピック以上のアイテムを扱う行商人が出現するようになる |
| Lv5（MAX） | 最大品揃えの行商人の出現率が確定。特殊行商人（設計図・種専門）が高確率で出現。改造オプション解放 |

## 改造オプション（Lv5 MAX後に解放）

> いずれか1つを選択。選択後は施設を解体しない限り変更不可。

### 高級市場
出現する行商人の品揃えが常に最高レベルに固定される。エピック以上の品物が定常的に並ぶようになり、最高品質の装備・素材を確実に入手できる機会が増える。出現確率は標準のまま。

### 常設市場
行商人の出現確率が最大化され、ほぼ毎回の帰還時に行商人に会えるようになる。コンスタントに特殊品を購入できる安定感が最高クラス。品揃えのレベルは標準的。

### 特殊市場
通常の行商人の代わりに「特殊行商人」（レアな設計図・貴重な種・ダンジョン限定素材などを専門に扱う）の出現率が大幅アップする。入手困難なレシピ解放・施設建設の設計図を揃えたい場合に最強。

## 備考
- 行商人の出現はランダムイベントであるため、「出会えなかった回」があってもゲームプレイに大きな支障が出ない設計を維持すること。
- 行商人の支払い通貨は「特殊コイン（ダンジョンで稀に入手）」「通常ゴールド」「素材交換」など複数の手段を想定。
