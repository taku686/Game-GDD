# 鑑定所

## 概要
ダンジョンで取得した「効果不明」「呪い」の可能性がある未鑑定アイテムの正体を判明させる施設。帰還後に真っ先に立ち寄る「帰還後処理」の起点。

## 基本情報

| 項目 | 内容 |
|---|---|
| **カテゴリ** | 帰還後処理 |
| **建設タイプ** | 1施設1つ固定（レベルアップ式） |
| **アンロック条件** | 鑑定士の霊魂をダンジョンで救出後にアンロック |

## 役割
ダンジョンから持ち帰った未鑑定アイテムを安全に使えるよう、その正体・効果・呪いの有無を明らかにする。鑑定時にはミニゲーム（間違い探しなど）が発生し、成功度に応じて鑑定費用の割引や追加バフの付与が得られる。

施設レベルが上がると鑑定の精度と速度が向上し、同時に鑑定できるアイテム数も増える。

## 霊魂配置

> 霊魂の配置・変更はこの施設で直接行うか、**役場**の一括管理UIからも行うことができる。受け入れ可能なスロット数は施設レベルによって決まる。

| 項目 | 内容 |
|---|---|
| **配置スロット数** | Lv1：1体 → Lv3：2体 → Lv5(MAX)：3体 |
| **最適な武器種** | 杖/魔導書（片手） |
| **武器の効果** | 鑑定速度アップ・鑑定後アイテムのレアリティ補正（上方修正）が入る確率アップ |
| **最適な防具** | 魔法衣（ローブなど） |
| **防具の効果** | 霊素との親和性が高く、大成功鑑定（追加バフ付き鑑定）の発生率が大幅アップ |
| **属性ボーナス** | 光属性の杖：呪われたアイテムの解呪成功率が大幅アップ。呪いの詳細な情報も開示される |
| **レアリティの影響** | 高レアリティの杖を持たせるほど、鑑定コスト削減率と「鑑定+エンチャント追加」の発生確率がアップ |

## レベルアップ効果

| レベル | 解放内容・強化内容 |
|---|---|
| Lv1 | 基本鑑定（コモン〜アンコモン品）が可能 |
| Lv2 | レア品の鑑定が解放。鑑定コストが低下 |
| Lv3 | 呪い解除の成功率アップ。霊魂配置スロット+1 |
| Lv4 | エピック品の鑑定が解放。同時鑑定数が増加 |
| Lv5（MAX） | 全レアリティの鑑定が可能。鑑定ミニゲームの難易度が低下。改造オプション解放 |

## 改造オプション（Lv5 MAX後に解放）

> いずれか1つを選択。選択後は施設を解体しない限り変更不可。

### 呪い専門所
呪い解除に完全特化した施設に転化する。呪い解除の成功率が大幅アップし、呪われたアイテムを安全に除呪するコストが大幅に削減される。呪いアイテムから逆に「呪いの効果を利用した強化」を行う特殊オプションも解放。通常の鑑定速度はやや低下する。

### 超鑑定所
鑑定の精度が飛躍的に向上する高品質路線。鑑定成功時に装備へ追加バフが付く確率が大幅アップする。最高成功時には「鑑定 + エンチャント付与 + レアリティ補正」が同時に得られる可能性がある。鑑定コストはやや上昇する。

## 備考
- 鑑定ミニゲームの詳細仕様は `09_ミニゲームシステム/` フォルダで管理予定。
- 未鑑定のまま装備・使用するとランダムな効果が発動するリスクがあるが、呪いアイテムを意図的に活用する「呪いビルド」の設計も視野に入れる。
- 「帰還後処理」の流れ：鑑定所 → 解体所（不要品の分解）→ 鍛冶屋（使う装備の強化）→ 保管庫（余剰品の保管）。
