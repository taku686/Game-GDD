# 解体所（リサイクル屋）

## 概要
ダンジョンで拾って余った不要な武器・防具を分解し、建築・強化用の素材に還元する施設。ハクスラ・トレハンで大量に得たアイテムを無駄なく活用するための帰還後処理施設。

## 基本情報

| 項目 | 内容 |
|---|---|
| **カテゴリ** | 帰還後処理 |
| **建設タイプ** | 1施設1つ固定（レベルアップ式） |
| **アンロック条件** | 役場をLv2以上にアップグレード後に建設可能 |

## 役割
大量に取得した不要な装備品を「鉱石」「木材」「魔法の粉」「特殊素材」などに分解・還元する。これにより「拾って使えなかったアイテム」も全て素材として活用でき、探索のすべての収穫に意味が生まれる。

素材加工所が「建築素材の変換」専門なのに対し、解体所は「装備品の分解」専門として役割が明確に分かれる。

## 霊魂配置

> 霊魂の配置・変更はこの施設で直接行うか、**役場**の一括管理UIからも行うことができる。受け入れ可能なスロット数は施設レベルによって決まる。

| 項目 | 内容 |
|---|---|
| **配置スロット数** | Lv1：1体 → Lv3：2体 → Lv5(MAX)：3体 |
| **最適な武器種** | 斧（片手） |
| **武器の効果** | 大型・重量のある装備（重鎧・両手武器・エピック）の分解効率が大幅に上昇する。鉱石・金属板など金属系素材の回収量が特に増加する |
| **最適な防具** | 軽鎧（レザーなど） |
| **防具の効果** | 素早い作業スピードで分解処理時間が短縮される |
| **属性ボーナス** | 地属性の斧：重量装備（重鎧・大型武器）の分解時に鉱石・金属板が追加で1個確定回収される。岩石・鉱物系素材の回収効率も全体的に上昇する |
| **レアリティの影響** | 高レアリティの斧を持たせるほど、分解時に得られる金属系素材のレアリティ底上げと量が増加する |

## レベルアップ効果

| レベル | 解放内容・強化内容 |
|---|---|
| Lv1 | コモン〜アンコモン装備の分解が可能。基本素材に還元 |
| Lv2 | レア装備の分解が解放。分解時の素材ロスが低下 |
| Lv3 | エンチャントの抽出が解放（装備のエンチャントを「魔法の粉」として回収）。霊魂配置スロット+1 |
| Lv4 | エピック装備の分解が解放。複数アイテムの同時分解が可能に |
| Lv5（MAX） | 全レアリティの分解が可能。分解時にレア素材を確定で1個追加獲得。改造オプション解放 |

## 改造オプション（Lv5 MAX後に解放）

> いずれか1つを選択。選択後は施設を解体しない限り変更不可。

### 精密解体所
分解時の素材回収率が最大化される。通常では一部が失われる素材も余さず回収でき、レア素材の抽出確率が大幅アップ。大事な装備を慎重に分解したい場合に最適。分解速度はやや遅くなる。

### 魔力分解所
装備に付与されたエンチャント（ルーン・特殊効果）を「魔法の粉」や「エンチャントクリスタル」として専門的に抽出する機能に特化。抽出したエンチャントは鍛冶屋で別の装備に付与できる（エンチャント移植）。通常の素材回収量はやや低下する。

### 大量解体工場
一度に処理できるアイテム数が大幅増加し、ハクスラで溜まった大量の装備品を一括解体できる。速度と処理量が特化されている分、各アイテムからの素材回収率は標準以下になる。量をさばくことが目的のプレイヤー向け。

## 備考
- エンチャント抽出（Lv3解放）は鍛冶屋との連携機能。鍛冶屋をアップグレードしていると使える素材の幅が広がる。
- 呪い装備の分解は通常の分解より高リスクだが、特殊素材を得られる可能性がある（施設レベル依存）。
- 「帰還後処理」の流れ：鑑定所 → 解体所（不要品の分解）→ 鍛冶屋（使う装備の強化）→ 保管庫（余剰品の保管）。
