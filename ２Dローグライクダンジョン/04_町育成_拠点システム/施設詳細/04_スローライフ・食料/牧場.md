# 牧場（モンスター飼育小屋）

## 概要
ダンジョンで捕獲・テイムした特定のモンスターや動物を飼育する施設。毎探索ごとに「卵」「ミルク」などの希少な食材を生産し、探索同行ペットの育成枠としても機能する。

## 基本情報

| 項目 | 内容 |
|---|---|
| **カテゴリ** | スローライフ・食料 |
| **建設タイプ** | 複数建設可能（最大3件） |
| **アンロック条件** | 最初のモンスターテイムに成功した後にアンロック |

## 役割
テイムした動物・モンスターを飼育して定期的に素材を生産する。生産される素材は料理の高品質食材として使えるほか、牧場に特化した施設では希少なモンスター素材も手に入る。

探索に連れて行く「ペット」の育成場としての機能も持ち、一緒に戦うペットを強化するための施設としても重要。

## 霊魂配置

> 霊魂の配置・変更はこの施設で直接行うか、**役場**の一括管理UIからも行うことができる。受け入れ可能なスロット数は施設レベルによって決まる。

| 項目 | 内容 |
|---|---|
| **配置スロット数** | Lv1：1体 → Lv3：2体 |
| **最適な武器種** | 爪/ナックル（片手） |
| **武器の効果** | モンスターの育成速度アップ・テイム済み魔物のステータスボーナス |
| **最適な防具** | 平服（町人服など） |
| **防具の効果** | モンスターとの信頼関係ボーナス（なつき度アップ速度が上昇） |
| **属性ボーナス** | 雷属性の爪：モンスターの特殊技（固有スキル）の習得速度が大幅アップ |
| **レアリティの影響** | 高レアリティの爪/ナックルを持たせるほど、牧場での特殊ドロップ（希少素材）の発生確率が上がる |

## レベルアップ効果

| レベル | 解放内容・強化内容 |
|---|---|
| Lv1 | 基本動物（鶏・羊など）の飼育が可能。卵・毛などの基本食材を生産 |
| Lv2 | 小型モンスターの飼育が解放。より珍しい素材を生産 |
| Lv3 | 中型モンスターの飼育が解放。ペットの育成機能が追加。霊魂配置スロット+1 |
| Lv4 | 大型・レアモンスターの飼育が解放。飼育数の上限が増加 |
| Lv5（MAX） | 特殊・最強クラスのモンスターも飼育可能。改造オプション解放 |

## 改造オプション（Lv5 MAX後に解放）

> いずれか1つを選択。選択後は施設を解体しない限り変更不可。
> 複数建設している場合は施設ごとに別の改造オプションを選べる。

### 希少種牧場
レアなモンスターの育成に特化した牧場に転換される。通常では手に入りにくい希少素材・特殊ドロップの生産量が大幅アップ。一般的な食材の生産量は低下するが、素材収集面での価値は最高クラス。

### ペット最適化牧場
探索に連れて行くペットの育成に特化した施設に転換される。ペットの能力値の上昇速度・特殊スキルの習得効率が大幅アップし、強力な同行ペットをより早く育成できる。食材生産は標準的な量を維持する。
