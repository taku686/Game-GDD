# 畑（農場）

## 概要
ダンジョンで拾った「種」を植え、作物を育てる食料生産施設。探索を重ねるごとに作物が育ち、収穫した食材は霊素（活動エネルギー）の回復や料理の素材に使う。

## 基本情報

| 項目 | 内容 |
|---|---|
| **カテゴリ** | スローライフ・食料 |
| **建設タイプ** | 複数建設可能（最大8件） |
| **アンロック条件** | ゲーム開始時から建設可能 |

## 役割
ダンジョン内での行動に伴い消費される「霊素（活動エネルギー）」を回復するための食材を安定供給する。畑を増やすほど生産量が増え、食料不足による探索の制約が減る。

探索回数（日数の経過）によって作物が育つ仕組みのため、「探索に行っている間に畑が育っている」というゲームサイクルとの相性が良い。

種の種類によって育てられる作物の種類・用途が異なる（薬草特化・食料特化など）。

## 霊魂配置

> 霊魂の配置・変更はこの施設で直接行うか、**役場**の一括管理UIからも行うことができる。受け入れ可能なスロット数は施設レベルによって決まる。

| 項目 | 内容 |
|---|---|
| **配置スロット数** | Lv1：1体 → Lv3：2体 |
| **最適な武器種** | 鎌（両手） |
| **武器の効果** | 収穫量アップ・作物の成長速度アップ |
| **最適な防具** | 軽鎧（レザーなど） |
| **防具の効果** | 素早い作業スピードで収穫サイクルが短縮される |
| **属性ボーナス** | 水属性の鎌：作物に自動的に水がまかれ、成長速度が劇的に上昇する |
| **レアリティの影響** | 高レアリティの鎌を持たせるほど、収穫時に「大収穫（2〜3倍ドロップ）」が発生する確率が上がる |

## レベルアップ効果

| レベル | 解放内容・強化内容 |
|---|---|
| Lv1 | 基本的な作物（野菜・果物）の栽培が可能 |
| Lv2 | 薬草類の栽培が解放。作物の成長速度が上昇 |
| Lv3 | 希少な作物の栽培が解放。霊魂配置スロット+1 |
| Lv4 | 作物の収穫量増加。複数種類の種を同時に植えられる枠が増加 |
| Lv5（MAX） | 最高品質の作物が育つようになる。改造オプション解放 |

## 改造オプション（Lv5 MAX後に解放）

> いずれか1つを選択。選択後は施設を解体しない限り変更不可。
> 複数建設している場合は畑ごとに別の改造オプションを選べる。

### 薬草畑
薬草・毒草などの調合素材に特化した農園に転換される。調合所で使う素材の品質と収穫量が大幅にアップする。食用作物は育てられなくなるが、消耗品の自給自足を目指す構成に強い。

### 高級農場
通常の食用作物を高品質化することに特化した農園に転換される。収穫した野菜・果物の品質が上がり、料理屋で調理した際のバフ効果値が増加する。薬草は育てられなくなる。

## 備考
- 畑の詳細な生産サイクル（成長速度・霊素回復量の数値など）は `04_町育成_拠点システム/食料生産とスローライフシステム.md` で管理（未着手）。
- 複数の畑を建てている場合、それぞれ異なる改造オプションを選べる。「薬草畑×3 ＋ 高級農場×5」のような特化配分も可能。
