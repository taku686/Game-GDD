# 釣り堀

## 概要
拠点の水場でミニゲームを通じて魚を釣る食料生産施設。釣った魚は料理の食材になるほか、レアな魚は高値での換金も可能。

## 基本情報

| 項目 | 内容 |
|---|---|
| **カテゴリ** | スローライフ・食料 |
| **建設タイプ** | 複数建設可能（最大3件） |
| **アンロック条件** | 役場をLv1以上にアップグレード後に建設可能 |

## 役割
ミニゲームを通じて魚（食材・換金アイテム）を獲得する。畑が「待つだけの自動生産」であるのに対し、釣り堀は「プレイヤーが能動的に関与する生産」として差別化される。

建設数が増えるほど釣れる魚の種類が増え、より深い水域（レア魚が出やすい）を選べるようになる。

## 霊魂配置

> 霊魂の配置・変更はこの施設で直接行うか、**役場**の一括管理UIからも行うことができる。受け入れ可能なスロット数は施設レベルによって決まる。

| 項目 | 内容 |
|---|---|
| **配置スロット数** | Lv1：1体 → Lv3：2体 |
| **最適な武器種** | 槍/ランス（両手） |
| **武器の効果** | 銛（もり）として機能。レアな大物・深海魚のヒット率・獲得品質がアップ |
| **最適な防具** | 軽鎧（レザーなど） |
| **防具の効果** | 素早いキャストができ、ミニゲームのアクション速度が上昇する |
| **属性ボーナス** | 水属性の槍：深海魚・魔界魚など特殊な魚の出現率が大幅アップ |
| **レアリティの影響** | 高レアリティの槍を持たせるほど、釣りミニゲームのヒット判定エリアが拡大しやすくなる |

## レベルアップ効果

| レベル | 解放内容・強化内容 |
|---|---|
| Lv1 | 基本的な淡水魚が釣れる |
| Lv2 | 海水魚・川魚など魚の種類が増加 |
| Lv3 | 深海エリアが解放（レア魚が出やすい）。霊魂配置スロット+1 |
| Lv4 | 大物魚・特殊魚が出現するようになる |
| Lv5（MAX） | 全魚種が釣れるようになる。改造オプション解放 |

## 改造オプション（Lv5 MAX後に解放）

> いずれか1つを選択。選択後は施設を解体しない限り変更不可。
> 複数建設している場合は施設ごとに別の改造オプションを選べる。

### 深海釣り堀
通常では出現しないレアな深海魚・魔界の魚（クリア後素材）が釣れるようになる。料理に使うと特別なバフが付与される高品質素材を確保できる。通常魚の出現率は低下する。

### 換金釣り堀
高値で売れる観賞魚・換金魚の出現率が大幅アップする。ゴールド収入を増やしたい時に活躍する。食材としての価値は低くなるが、資金調達の重要な手段になる。
