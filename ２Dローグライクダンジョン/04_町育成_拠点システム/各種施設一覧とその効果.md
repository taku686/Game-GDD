# 施設一覧とその役割

本ドキュメントでは、拠点の町に建設・拡張できる各種施設と、それぞれのゲーム内での役割、効果について定義します。
ダンジョンから持ち帰った「素材」と「資金」を使って施設を建設・アップグレードすることで、次回以降の探索が恒久的に有利になります（メタプログレッション）。

## 1. 探索準備・装備関連

### 武器屋・防具屋
*   **概要**: 探索前に持ち込める初期装備（武器・防具）を販売する施設。
*   **役割**: 資金（ゴールド）を消費して基本的な装備を整える。施設のアップグレードにより、品揃えが増えたり、初期から強力なエンチャントが付与された装備が並ぶようになる。

### 道具屋
*   **概要**: ポーション（回復薬）、帰還アイテム（脱出用）、松明などの消費アイテムを販売する施設。
*   **役割**: 探索中の生存率を高めるための基本物資調達。持ち込み数には制限があるため、何を買い揃えるかが重要。拠点レベルが上がると上位の回復薬が購入可能に。

### 鍛冶屋
*   **概要**: 装備の強化、修理、および特殊効果（ルーンなど）の付与を行う施設。
*   **役割**: ダンジョンから持ち帰った鉱石などの素材を使って、お気に入りの装備を強化する。
*   **特記事項**: 強化の際には「ミニゲーム」が発生し、プレイヤーの腕前（スコア）によって強化の成功度合いや付与されるステータスボーナスが変動する。

### 鑑定所
*   **概要**: ダンジョンで取得した「効果不明」や「呪い」の可能性がある未鑑定アイテムの正体を判明させる施設。
*   **役割**: 未鑑定のまま使用するリスクを排除する。
*   **特記事項**: 鑑定時にはミニゲーム（間違い探しなど）が発生し、進行度や成功度によって鑑定費用の割引や追加のバフが付与される。

### 調合所（錬金術アトリエ）
*   **概要**: 薬草類やモンスターのドロップ素材を使って、戦闘用の消耗品を作成・合成する施設。
*   **役割**: 料理とは異なる「強力な回復ポーション」「爆弾」「状態異常回復薬」などを作成。また、下位素材を上位素材に変換する機能も持つ。

### 解体所（リサイクル屋）
*   **概要**: ダンジョンで大量に拾って余った不要な武器・防具を分解する施設。
*   **役割**: 不要品を「鉱石」「木材」「魔法の粉」などの建築・強化素材に還元し、ハクスラ・トレハンで得たアイテムを無駄なく活用する。

## 2. キャラクター強化・管理関連

### 転職所（ギルド）
*   **概要**: プレイヤーキャラクターのクラス（職業）の変更、および永続的なパッシブスキルの解放を行う施設。
*   **役割**: 攻略するダンジョンの特性に合わせた職業選択を行う。施設のレベルアップや特定のクエストクリアによって、新たな「上位職」がアンロックされる。

### 保管庫（倉庫）
*   **概要**: ダンジョンから無事に持ち帰ったアイテムや素材を預けておくことができる施設。
*   **役割**: ゲームオーバー時（ダンジョンでの死亡時）の全ロスを防ぐための安全な資産保管場所。アップグレードすることで保管できるアイテムの枠（スロット数）が増加する。

### 祈祷所（神殿 / 祠）
*   **概要**: 探索に出発する前に、資金やアイテムを奉納してランダムなバフを受ける施設。
*   **役割**: 「ドロップ率アップ」「獲得経験値増加」「罠・呪い耐性アップ」などの効果を次回の探索時のみ付与し、生存率や期待値を高める。

### 転移門（ワープ陣）
*   **概要**: すでに攻略済みの深い階層へ直接ワープできるショートカット施設。
*   **役割**: 浅い階層の探索をスキップし、効率よく深層の攻略や高難度のアイテム収集に向かうための時間短縮要素。施設レベルでワープ先の選択肢が増加する。

## 3. スローライフ・食料生産（霊素・スタミナ対策）

ダンジョン内での時間経過や行動に伴って消費される「活動エネルギー（霊素）」を回復（吸収）するための重要な生産サイクルを担います。

### 畑（農場）
*   **概要**: ダンジョンで拾った「種」を植え、作物を育てる施設。
*   **役割**: 霊素を満たすための基本的な「食材」の安定供給。日数の経過（探索を重ねる）によって作物が育つ。

### 釣り堀
*   **概要**: 拠点の水場で魚を釣ることができる施設。
*   **役割**: ミニゲームを通じて魚（食材や換金アイテム）を獲得する。釣った魚は料理の素材になるほか、レアな魚は高値で売却可能。

### 牧場（モンスター飼育小屋）
*   **概要**: ダンジョンで捕獲・テイムした特定のモンスターや動物を飼育する施設。
*   **役割**: 毎日（探索ごと）に「卵」や「ミルク」などの貴重な食材を生産する。また、一緒に探索へ連れて行く「ペット」の育成枠としての機能も持たせる。

### 料理屋（食堂）
*   **概要**: 畑・釣り堀・牧場で集めた「食材」を使って「料理」を作る施設。
*   **役割**: 生の食材から吸収するよりも霊力の回復量が飛躍的に上がるだけでなく、「最大HPアップ」「攻撃力アップ」などの**探索中永続するバフ**を付与するアイテムを作成できる。

## 4. 町の発展・システム関連

### 役場（建築所）
*   **概要**: 町全体の管理と、新たな施設の建設・アップグレードを統括する中心施設。
*   **役割**: ダンジョンから持ち帰った「建築素材（木材、石材など）」と資金を消費し、他のすべての施設をレベルアップさせる。町全体の発展レベル（メタプログレッション）の要。

### 酒場 / クエストボード
*   **概要**: 霊魂たち（NPC）が集まる情報交換の場であり、様々な依頼（サブクエスト）を受注できる施設。
*   **役割**: 「特定のモンスターを〇匹討伐」「〇〇というアイテムを納品」などのクエストをこなし、報酬としてレア素材や新しい施設の「設計図」を獲得する。

### 闇市（行商人のテント）
*   **概要**: ダンジョンから帰還するたびに品揃えがランダムに変化する特殊なショップ。
*   **役割**: 通常の店舗（武器屋・道具屋）には並ばない激レア装備、特別な種、変わった設計図などを高額あるいは特殊な通貨で販売する。

### 博物館（図書館 / 記録所）
*   **概要**: 遭遇したモンスター、拾ったアイテム、釣った魚などの「図鑑」を展示・記録していく施設。
*   **役割**: プレイヤーのやり込み（コレクション）要素。図鑑の完成度が上がるごとに、町全体や全キャラクターへの恒久的なパッシブバフ（ステータス底上げ）が得られる。

### 探検隊本部（傭兵ギルド）
*   **概要**: 使っていない別クラスのキャラクターや、牧場のペットたちを「自動素材集め」に派遣するシステム。
*   **役割**: プレイヤーがメインキャラクターでダンジョンを探索している裏で、自動的にアイテムや素材を収集してくれる放置・自動化要素。
