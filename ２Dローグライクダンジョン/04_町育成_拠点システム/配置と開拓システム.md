# 施設の自由配置と開拓システム

単に一覧の施設を順番にレベルアップさせるだけでなく、**プレイヤーが自分だけの町をデザインし、少しずつ領地を広げていく**サンドボックス的な要素と、資源を集めて発展させるサバイバルゲーム要素を取り入れます。

## 1. リアルタイム建築システム（ホワイトサバイバル風）
**「資源の調達はダンジョンで、建築はリアルタイム進行」**というシステムを採用し、プレイのメリハリを生み出します。

*   **ダンジョンでの素材調達**
    建物の建設やアップグレードには、木材や石材、特殊な金属などが必要です。これらは主にダンジョンへ潜ることで収集します。レベルの高い施設であるほど、より深層でしか手に入らないレア素材が要求されます。
*   **建築完了までのリアルタイム経過**
    建設やアップグレードを指示すると、完了までに「リアルタイムの待ち時間（例: 数分～数時間）」が発生します。
*   **充実した待ち時間の活用**
    建築を待つ時間が退屈にならないよう、以下のようなアクションで時間を有効活用できるゲームサイクルにします。
    *   **スキマ時間でのダンジョン探索**: 建築を進めている間に次の階層向けの素材や資金を求めてダンジョンへ潜る。
    *   **ミニゲームのプレイ**: 拠点での釣り堀や畑の世話、鍛冶屋でのミニゲームなどをこなし、食料やバフを蓄える。
    *   **自由な配置と都市計画**: 下記で説明する「自由配置システム」を使って、次に何をどこに建てるかを考え、建設中の町並みやレイアウトをいじる時間として楽しむ。

## 2. 施設の自由配置と選択
町のマップ（グリッド制を想定）において、プレイヤーが好きな場所に、好きな施設を選んで建設・配置できるシステムです。

*   **必須施設と任意施設**
    *   「役場」などゲーム進行に不可欠なものは初期配置されますが、それ以外の施設（お店、畑、牧場など）は、解放済みの土地の好きな場所に建設できます。
*   **複数建設による特化**
    *   「畑」を大量に並べて農園特化にしたり、「保管庫」を複数建ててアイテム備蓄を増やすなど、プレイヤーごとのプレイスタイルが町づくりに反映されます。
*   **レイアウトのカスタマイズ**
    *   一度建てた施設も移動や撤去ができ、建設待ちの時間を使って「自分なりの導線や見栄えの良い町並み」をじっくり考えることができます。

## 3. 開拓システム（未開の地の解放）
ゲーム開始時、町の大部分は「未開の地」（深い森、巨大な岩場、崩落したガレキなど）に覆われており、施設を建てられるスペースはごくわずかです。

*   **土地を解放（開拓）する条件**
    以下の条件などで建築エリアを拡大できます。
    1.  **ダンジョン進行度**: 特定のボスを倒す。
    2.  **人口の増加**: ダンジョンで霊魂を救出する。
    3.  **特定の施設やツールの解放**: 「伐採所」を建てたり、ダンジョンで「丈夫なツルハシ」を見つける。
    4.  **資源の消費（工事）**: 集めた資金や素材を消費して瓦礫を撤去する（ここでも待ち時間が発生する想定）。
*   **お宝の発見（開拓ボーナス）**
    未開の地を取り除く際「レアな建築素材」「隠し資金（宝箱）」「新しい建築物の設計図」などが発見されるボーナス要素を用意します。

## 4. ゲームサイクルの中での役割まとめ
このシステムにより、プレイヤーは常に「次にやること」がある状態を作れます。

1.  **ダンジョン探索**：建築素材、設計図、資金を集め、霊魂を救出する。
2.  **建築・開拓の指示**：素材を使って施設の建設やエリアの開拓を指示する（リアルタイムで時間がかかる）。
3.  **待ち時間の活用**：建築が終わるまでの間に、別のダンジョンへ潜ったり、ミニゲーム（釣りや畑）で準備を整えたり、町のレイアウトを考える。
4.  **恩恵の獲得と再挑戦**：完成した施設で恒久的なバフや強力なアイテムを得て、さらに深いダンジョンへ挑戦し、よりレアな素材を持ち帰る。
